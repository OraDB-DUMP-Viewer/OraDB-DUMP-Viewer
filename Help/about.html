バージョン確認

メニューの ヘルプ(&H) > バージョン情報(&A) で 現在のバージョンを確認できます。

バージョン情報ダイアログでは、GitHub の最新リリースと比較して アップデートの有無が自動的に表示されます。

  • 最新版を使用中の場合: バージョン番号が緑色で表示されます
  • 新しいバージョンがある場合: バージョン番号がオレンジ色で表示されます

自動更新

新しいバージョンが利用可能な場合、ダイアログ内に更新ボタンが表示されます。

  1. バージョン情報ダイアログの 最新バージョンに更新する ボタンをクリックします。
  2. 確認ダイアログで はい を選択します。
  3. 最新の MSI インストーラーが自動的にダウンロードされます (進捗バーが表示されます)。
  4. ダウンロード完了後、アプリケーションが終了し、インストーラーが自動起動します。

また、リリースページを開く リンクから GitHub のリリースページを ブラウザで確認することもできます。

ライセンスはローカルでの RSA 署名検証に加え、起動時にオンラインでの有効性検証が行われます。 オフライン環境では従来通りローカル検証のみで利用可能です。

ライセンス認証は ヘルプ(&H) > ライセンス認証 から 再認証やライセンスキーの変更が行えます。

エラー報告

バグや問題を見つけた場合は、ヘルプ(&H) > エラー報告(&R) から報告できます。

エラー報告ダイアログでは以下の情報が自動収集され、透明に表示されます:

  • アプリケーションバージョン / DLL バージョン
  • OS バージョン / .NET バージョン
  • アーキテクチャ / 画面解像度 / DPI
  • ダンプファイル情報 (サイズ・形式のみ、パスは送信されません)

報告は Cloudflare Worker 経由で GitHub Issues に自動作成されます。 個人情報 (ユーザー名、PC名、ファイルパス等) は収集・送信されません。

ダンプファイルが 50MB 以下の場合は、任意でダンプファイルを添付することもできます。

多言語対応

日本語・英語・中国語(簡体)・韓国語・ドイツ語・フランス語・スペイン語・イタリア語・ロシア語・ポルトガル語(BR) の10言語に対応しています。 言語はアプリケーション起動時にシステムの表示言語に合わせて自動選択されます。 ヘルプメニューの「言語」から手動で切り替えることもできます。
