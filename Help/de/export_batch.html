Stapelexport

Übersicht

Exportieren Sie mehrere Tabellen gleichzeitig in CSV- oder SQL-Dateien. Jede Tabelle wird als einzelne Datei ausgegeben.

Vorgehensweise

  1. Klicken Sie auf die Schaltfläche Stapelexport in der Symbolleiste.
  2. Der Exportdialog wird angezeigt.
  3. Wählen Sie das Ausgabeformat (CSV / SQL).
  4. Geben Sie den Ausgabeordner an.
  5. Bestätigen Sie die zu exportierenden Tabellen.
    Deaktivieren Sie die Kontrollkästchen für nicht benötigte Tabellen.
  6. Klicken Sie auf die Schaltfläche Exportieren.

Ausgabedateinamen

Dateinamen werden automatisch im Format Schemaname.Tabellenname.csv oder Schemaname.Tabellenname.sql generiert.

Tabellenauswahl / -ausschluss

  • Schaltflächen Alle auswählen / Alle abwählen für Massenoperationen
  • Auswahl umkehren zum Umschalten der Auswahlzustände
  • Klicken Sie auf einzelne Kontrollkästchen, um bestimmte Tabellen auszuwählen
Tipp: Im Arbeitsbereich ausgeschlossene Tabellen werden automatisch auch vom Stapelexport ausgeschlossen.

Berichtsausgabe

Über Objekt(&O) > Berichtsausgabe(&R) können Sie auch Tabelleninformationen in Textdateien ausgeben.

  • Objektliste — Textliste der Tabellennamen und Zeilenanzahlen ausgeben
  • Tabellendefinitionen — Spaltendefinitionen (Name, Typ, Größe) für alle Tabellen ausgeben
