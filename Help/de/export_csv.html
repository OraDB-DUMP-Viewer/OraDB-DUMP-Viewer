CSV-Export

Vorgehensweise

  1. Wählen Sie die zu exportierende Tabelle in der Tabellenliste aus.
  2. Klicken Sie auf die Schaltfläche CSV in der Symbolleiste, oder wählen Sie Objekt(&O) > Export > Daten(&D).
  3. Geben Sie den Zieldateipfad an.
  4. Der Export wird ausgeführt und ein Fortschrittsdialog wird angezeigt.

Ausgabeformat

  • Kodierung: UTF-8 (mit BOM)
  • Trennzeichen: Komma (,)
  • Anführungszeichen: Doppelte Anführungszeichen (") — werden angewendet, wenn Werte Kommas, Zeilenumbrüche oder doppelte Anführungszeichen enthalten
  • Zeilenende: CRLF (\r\n)
  • RFC 4180 konform

Optionen

Folgendes kann in den Exportoptionen konfiguriert werden:

  • Kopfzeile — Ob Spaltennamen in der ersten Zeile ausgegeben werden sollen
  • Typinformationszeile — Spaltentyp-Informationen in der Zeile nach der Kopfzeile ausgeben
  • Datumsformat — Ausgabeformat für DATE/TIMESTAMP-Spalten

Weitere Exportformate

Excel-Export

Exportiert Tabellendaten in eine Excel-Datei (.xlsx / .xls).

  1. Wählen Sie eine Tabelle aus und klicken Sie auf die Schaltfläche Excel in der Symbolleiste.
  2. Geben Sie den Zieldateipfad an.
  3. Eine Excel-Datei wird erstellt.

Access-Export

Exportiert Tabellendaten in eine Microsoft Access-Datenbank (.accdb / .mdb).

  1. Wählen Sie eine Tabelle aus und klicken Sie auf die Schaltfläche Access in der Symbolleiste.
  2. Geben Sie den Zieldateipfad an.
  3. Eine Tabelle wird in der Access-Datenbank erstellt und die Daten werden importiert.

SQL Server-Export

Exportiert Tabellendaten direkt in eine SQL Server-Datenbank.

  1. Wählen Sie eine Tabelle aus und klicken Sie auf die Schaltfläche SQL Server in der Symbolleiste.
  2. Geben Sie die SQL Server-Verbindungsinformationen im Verbindungsdialog ein.
  3. Eine Tabelle wird in der Zieldatenbank erstellt und die Daten werden eingefügt.

Wenn bereits eine Tabelle mit demselben Namen existiert, wird sie automatisch gelöscht und neu erstellt.

ODBC-Export

Exportiert Tabellendaten in eine beliebige ODBC-kompatible Datenbank.

  1. Wählen Sie eine Tabelle aus und klicken Sie auf die Schaltfläche ODBC in der Symbolleiste.
  2. Geben Sie die ODBC-Verbindungszeichenkette oder den DSN im Verbindungsdialog ein.
  3. Eine Tabelle wird in der Zieldatenbank erstellt und die Daten werden eingefügt.

Wenn bereits eine Tabelle mit demselben Namen existiert, wird sie automatisch gelöscht und neu erstellt.

Umgang mit vorhandenen Tabellen

ODBCAutomatisches Löschen + Neuerstellen

Hinweis: Für SQL Server- und ODBC-Exporte muss die Zieldatenbank vorab erstellt werden.
