Exportoptionen

Zugriff

Klicken Sie auf die Schaltfläche Optionen in der Symbolleiste, um den Dialog für Exportoptionen zu öffnen.

Datumsformat

FormatBeispielBeschreibung
YYYY/MM/DD HH:MI:SS2026/01/15 09:30:00Standardformat (Standard)
YYYYMMDD20260115Kompakt (nur Datum)
YYYYMMDDHH24MISS20260115093000Kompakt (Datum und Uhrzeit)
BenutzerdefiniertBeliebigToken: YYYY, MM, DD, HH24, MI, SS

CSV-Optionen

EinstellungStandardBeschreibung
Kopfzeile ausgebenEINSpaltennamen in der ersten Zeile ausgeben
Typinformationszeile ausgebenAUSSpaltentypen in der Zeile nach der Kopfzeile ausgeben

SQL-Optionen

Vor dem SQL-Export wird ein Dialog angezeigt. Folgende Optionen können per Kontrollkästchen umgeschaltet werden:

EinstellungStandardBeschreibung
DROP TABLE + CREATE TABLE ausgebenEINDROP TABLE IF EXISTS und CREATE TABLE DDL vor INSERT-Anweisungen ausgeben. Per Kontrollkästchen umschaltbar. Löscht und erstellt vorhandene Tabellen neu
DBMS-TypOracleOracle / PostgreSQL / MySQL / SQL Server
Hinweis: Optionseinstellungen werden in den Anwendungseinstellungen gespeichert und bleiben nach einem Neustart erhalten.
