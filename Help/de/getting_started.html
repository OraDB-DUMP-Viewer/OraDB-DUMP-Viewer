Was ist OraDB DUMP Viewer?

OraDB DUMP Viewer ist ein eigenständiges Werkzeug, mit dem Sie Oracle-Datenbank-Dump-Dateien (EXP / EXPDP) ohne Oracle-Client durchsuchen und exportieren können.

Hauptfunktionen

  • Dump-Datei-Analyse — Unterstützt sowohl EXP (Legacy) als auch EXPDP (Data Pump) Formate
  • Tabellenliste — Zeigt Schemas, Tabellennamen und Spalteninformationen in einer Baumansicht an
  • Datenvorschau — Seitenweise Datenanzeige mit Sortierung und Kopierfunktion
  • Erweiterte Suche — Daten filtern mit 12 Operatoren und UND/ODER-Verknüpfungen
  • CSV-Export — Tabellendaten in CSV-Dateien exportieren
  • SQL-Export — INSERT-Anweisungen generieren (Oracle / PostgreSQL / MySQL / SQL Server)
  • Excel-Export — Export in .xlsx / .xls Format
  • Access-Export — Export in .mdb / .accdb Datenbanken
  • SQL Server-Export — Direkter Export in eine SQL Server-Datenbank
  • ODBC-Export — Export in jede ODBC-kompatible Datenbank
  • Stapelexport — Mehrere Tabellen gleichzeitig nach CSV/SQL exportieren
  • LOB-Dateiextraktion — BLOB/CLOB/NCLOB-Daten als einzelne Dateien extrahieren
  • Arbeitsbereich — Arbeitszustand speichern und wiederherstellen
  • Tabellenausschluss — Unerwünschte Tabellen ausblenden (mit Rückgängig/Wiederherstellen-Unterstützung)
  • Mehrsprachige Unterstützung — 10 Sprachen einschließlich Japanisch, Englisch, Chinesisch, Koreanisch, Deutsch, Französisch, Spanisch, Italienisch, Russisch und Portugiesisch (BR)
  • Fehlerberichte — Fehler direkt aus der App an GitHub Issues melden
  • Automatische Aktualisierung — Erkennt neue Versionen und bietet Ein-Klick-Updates

Unterstützte Dump-Formate

FormatWerkzeugUnterstützungsstatus
EXP (Legacy)exp / impUnterstützt (Oracle 7–11g)
EXPDP (Data Pump)expdp / impdpUnterstützt (Oracle 10g–23ai)
EXPDP komprimiertexpdp COMPRESSION=ALLTeilweise (nur Tabellendefinitionen)

Installation

  1. Führen Sie den mitgelieferten MSI-Installer aus.
  2. Folgen Sie den Anweisungen des Installationsassistenten.
  3. Starten Sie „OraDB DUMP Viewer“ über den Desktop oder das Startmenü.
