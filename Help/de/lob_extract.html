LOB-Dateiextraktion

Übersicht

Extrahiert Daten aus BLOB / CLOB / NCLOB-Spalten einer Tabelle als eine Datei pro Datensatz.

Unterstützte Datentypen

SpaltentypAusgabeinhalt
BLOBBinärdateien (Bilder, PDF, ZIP usw.)
CLOBTextdateien (dekodiert mit dem Zeichensatz der Dump-Datei)
NCLOBUnicode-Textdateien

Vorgehensweise

  1. Wählen Sie eine Tabelle mit LOB-Spalten in der Tabellenliste aus.
  2. Wählen Sie Datei(&F) > Dateien extrahieren(&F) aus dem Menü.
  3. Der LOB-Extraktionsdialog wird angezeigt.
  4. Konfigurieren Sie Folgendes:
    • LOB-Spalte — Wählen Sie die zu extrahierende LOB-Spalte aus
    • Ausgabeordner — Verzeichnis für die Ausgabedateien
    • Dateibenennung — Fortlaufende Nummerierung oder Spaltenwert
    • Erweiterung — Dateierweiterung für Ausgabedateien (Standard: lob)
  5. Klicken Sie auf die Schaltfläche Extrahieren.

Dateibenennungsmethoden

MethodeBeispielBeschreibung
Fortlaufend000001.jpg, 000002.jpg6-stellige fortlaufende Nummern ab 1
Spaltenwertdateiname_spalte_wert.jpgVerwendet den Wert einer angegebenen Spalte als Dateinamen

Anwendungsbeispiele

Bilddateien extrahieren

Wenn JPEG-Bilder in einer BLOB-Spalte gespeichert sind:

  1. LOB-Spalte: IMAGE_DATA (BLOB) auswählen
  2. Erweiterung: jpg
  3. Dateibenennung: Fortlaufend oder ID-Spaltenwert

Textdaten extrahieren

Wenn XML oder JSON in einer CLOB-Spalte gespeichert ist:

  1. LOB-Spalte: XML_CONTENT (CLOB) auswählen
  2. Erweiterung: xml
Achtung: Die Extraktion kann bei vielen LOB-Datensätzen einige Zeit dauern. Sie können den Vorgang über den Fortschrittsdialog abbrechen.
Hinweis: In der Tabellenvorschau werden LOB-Daten als Platzhalter angezeigt: %BLOB%, %CLOB%, %NCLOB%. Verwenden Sie diese LOB-Dateiextraktionsfunktion, um die tatsächlichen Daten abzurufen.
