Eine Dump-Datei öffnen

Vorgehensweise

  1. Wählen Sie Datei(&F) > Dump-Datei öffnen(&O)... aus dem Menü.
    Oder klicken Sie auf die Schaltfläche Öffnen in der Symbolleiste.
  2. Wählen Sie im Dateidialog eine Dump-Datei (.dmp) aus.
  3. Die Analyse beginnt und ein Fortschrittsbalken wird angezeigt.
  4. Nach Abschluss erscheint die Tabellenliste im Arbeitsbereich.

Automatische Erkennung

Das Dump-Dateiformat (EXP / EXPDP / EXPDP komprimiert) wird automatisch durch Analyse der führenden Bytes und Markierungsbytes der Datei erkannt.

FormatErkennungsmethode
EXPFührende Bytes 0x01-0x05 + „EXPORT:“-Zeichenkette
EXPDP„xml version“-Zeichenkette
EXPDP (komprimiert)„KGC“ + „HDR“-Markierungen

Zeichensatz

Der Zeichensatz der Dump-Datei wird ebenfalls automatisch erkannt. Unterstützte Zeichensätze:

  • AL32UTF8 / UTF8
  • JA16SJIS (Shift_JIS)
  • JA16EUC (EUC-JP)
  • AL16UTF16
  • US7ASCII
  • WE8ISO8859P1 / WE8MSWIN1252
Hinweis: Bei komprimierten EXPDP-Dumps können nur Tabellendefinitionen (DDL) gelesen werden. Die Dekomprimierung von Datensätzen wird nicht unterstützt.
