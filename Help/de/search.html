Suche & Filter

Tabellennamenfilter

Geben Sie einen Teil des Tabellennamens in das Suchtextfeld oben im Arbeitsbereich ein, um die Tabellenliste in Echtzeit zu filtern.

  • Groß-/Kleinschreibung wird nicht unterschieden.
  • Teilweise Übereinstimmung bei Tabellennamen.
  • Löschen Sie den Text, um den Filter zu entfernen.

Erweiterte Suche (Tabellendatensuche)

Im Tabellenvorschau-Bildschirm können Sie erweiterte bedingungsbasierte Suchen in den Daten durchführen. Klicken Sie auf die Schaltfläche Erweiterte Suche, um den Suchbedingungsdialog zu öffnen.

Suchbedingungen festlegen

Geben Sie für jede Bedingungszeile Folgendes an:

  1. Spaltenname — Wählen Sie die Zielspalte aus
  2. Operator — Wählen Sie aus 12 Operatoren
  3. Suchwert — Geben Sie den Vergleichswert ein (nicht erforderlich bei Null / Nicht Null Operatoren)
  4. Groß-/Kleinschreibung beachten — Aktivieren für Vergleich mit Berücksichtigung der Groß-/Kleinschreibung

Zusammengesetzte Bedingungen (AND / OR)

Beim Hinzufügen von Bedingungen können Sie den logischen Operator dazwischen als AND oder OR festlegen.

  • AND — Nur Zeilen anzeigen, die alle Bedingungen erfüllen
  • OR — Zeilen anzeigen, die eine der Bedingungen erfüllen

Bedingungszeilen können frei hinzugefügt oder entfernt werden, was eine komplexe Filterung ermöglicht.

Suchbedingungen beibehalten

Suchbedingungen werden beim erneuten Öffnen derselben Tabelle automatisch wiederhergestellt. Um Bedingungen zu löschen, verwenden Sie die Schaltfläche Suche löschen im Vorschaubildschirm.

Auswahl umkehren

Sie können den Auswahlzustand der Tabellenliste umkehren. Verwenden Sie Auswahl umkehren aus dem Kontextmenü.
