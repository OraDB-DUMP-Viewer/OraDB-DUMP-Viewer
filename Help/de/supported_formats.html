Einschränkungen

  • Datensätze in komprimierten EXPDP-Dumps können nicht gelesen werden (nur Metadaten)
  • Tatsächliche Daten des BFILE-Typs können nicht extrahiert werden (nur Pfadinformationen)
  • Verschlüsselte Dump-Dateien werden nicht unterstützt
  • Dumps über Network Link werden nicht unterstützt
  • Maximale Anzahl von Tabellen: 1.000
  • Maximale Anzahl von Spalten: 1.000 (pro Tabelle)

Zeichensätze

Folgende NLS-Zeichensätze werden unterstützt:

  • AL32UTF8 / UTF8
  • JA16SJIS
  • JA16EUC
  • AL16UTF16
  • US7ASCII
  • WE8ISO8859P1 / WE8MSWIN1252
