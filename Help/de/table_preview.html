Tabellenvorschau

Eine Tabelle öffnen

Doppelklicken Sie auf einen Tabellennamen in der Tabellenliste, um die Datenvorschau zu öffnen. Oder klicken Sie mit der rechten Maustaste auf die Tabelle und wählen Sie Öffnen(&O).

Datenanzeige

Daten werden in einem DataGridView angezeigt.

  • Spaltennamen erscheinen in den Spaltenüberschriften.
  • Jede Zeile zeigt einen Datensatz an.
  • NULL-Werte werden als leer dargestellt.
  • LOB-Daten (BLOB/CLOB/NCLOB) werden als %BLOB%, %CLOB%, %NCLOB% angezeigt.

Seitennavigation

Bei großen Datenmengen werden die Daten seitenweise angezeigt.

  • Zeilen pro Seite — Einstellbar über das Zahleneingabefeld oben rechts (Standard: 100 Zeilen)
  • Seiteninformation — Aktuelle Seitennummer und Gesamtseitenzahl werden angezeigt
  • Verwenden Sie die Navigationsschaltflächen zum Seitenwechsel

Sortierung

Klicken Sie auf eine Spaltenüberschrift, um die Daten nach dieser Spalte zu sortieren. Klicken Sie erneut, um zwischen aufsteigender und absteigender Reihenfolge zu wechseln.

Zellen kopieren

Wählen Sie Zellen in der Vorschau aus und drücken Sie Ctrl+C, um sie in die Zwischenablage zu kopieren. Mehrfachauswahl und Kopieren werden ebenfalls unterstützt.

Erweiterte Suche

Klicken Sie auf die Schaltfläche Erweiterte Suche im Vorschaubildschirm, um Daten mit 12 Operatoren und UND/ODER-Verknüpfungen zu filtern. Siehe Suche & Filter für Details.

Export aus der Vorschau

Folgende Exporte sind über die Vorschau-Symbolleiste verfügbar:

  • SQL — SQL-Skriptausgabe
  • CSV — CSV-Dateiausgabe
  • Excel — Excel-Dateiausgabe
  • Access — Access-Datenbankausgabe
  • SQL Server — Direktausgabe an SQL Server
  • ODBC — Ausgabe an eine ODBC-kompatible Datenbank

Tabelleneigenschaften

Klicken Sie mit der rechten Maustaste auf eine Tabelle und wählen Sie Eigenschaften(&P), um Spaltendefinitionen (Spaltenname, Datentyp, Länge usw.) anzuzeigen.

Tabellenausschluss

Sie können unerwünschte Tabellen vorübergehend ausblenden.

  1. Wählen Sie Tabellen in der Tabellenliste aus (Mehrfachauswahl möglich).
  2. Wählen Sie Löschen(&D) aus dem Kontextmenü, oder drücken Sie die Entf-Taste.
  3. Ausgeschlossene Tabellen werden in der Statusleiste angezeigt.

Um den Ausschluss rückgängig zu machen:

  • Bearbeiten(&E) > Rückgängig (Ctrl+Z)
  • Objekt(&O) > Alle Filter löschen(&F) um alle Ausschlüsse zu entfernen
