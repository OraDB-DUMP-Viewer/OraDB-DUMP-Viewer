Checking the Version

Select Help(&H) > About(&A) from the menu to check the current version.

The About dialog automatically compares with the latest GitHub release and shows whether an update is available.

  • If using the latest version: the version number is displayed in green
  • If a newer version is available: the version number is displayed in orange

Auto Update

When a new version is available, an update button appears in the dialog.

  1. Click the Update to Latest Version button in the About dialog.
  2. Select Yes in the confirmation dialog.
  3. The latest MSI installer is downloaded automatically (a progress bar is displayed).
  4. After download, the application closes and the installer launches automatically.

You can also click the Open Release Page link to view the GitHub release page in your browser.

Bug Reports

If you find a bug or issue, report it from Help(&H) > Bug Report(&R).

The following information is automatically collected and transparently displayed in the report dialog:

  • Application version / DLL version
  • OS version / .NET version
  • Architecture / Screen resolution / DPI
  • Dump file info (size and format only; paths are not sent)

Reports are automatically submitted as GitHub Issues via a Cloudflare Worker. No personal information (username, PC name, file paths, etc.) is collected or sent.

If the dump file is 50 MB or less, you can optionally attach it to the report.

Multilingual Support
