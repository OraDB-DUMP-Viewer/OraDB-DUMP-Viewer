Batch Export

Overview

Export multiple tables to CSV or SQL files at once. Each table is output as an individual file.

Steps

  1. Click the Batch Export button on the toolbar.
  2. The export dialog is displayed.
  3. Select the output format (CSV / SQL).
  4. Specify the output folder.
  5. Confirm the tables to export.
    Uncheck any tables you don't need.
  6. Click the Export button.

Output File Names

File names are automatically generated in the format SchemaName.TableName.csv or SchemaName.TableName.sql.

Table Selection / Exclusion

  • Select All / Deselect All buttons for bulk operations
  • Invert Selection to toggle selection states
  • Click individual checkboxes to select specific tables
Tip: Tables excluded in the workspace are automatically excluded from batch export as well.

Report Output

From Object(&O) > Report Output(&R), you can also output table information to text files.

  • Object List — Output a text list of table names and row counts
  • Table Definitions — Output column definitions (name, type, size) for all tables
