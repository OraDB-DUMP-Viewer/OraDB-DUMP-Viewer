CSV Export

Steps

  1. Select the table you want to export in the table list.
  2. Click the CSV button on the toolbar, or select Object(&O) > Export > Data(&D).
  3. Specify the destination file path.
  4. The export runs and a progress dialog is displayed.

Output Format

  • Encoding: UTF-8 (with BOM)
  • Delimiter: Comma (,)
  • Quoting: Double quotes (") — applied when values contain commas, newlines, or double quotes
  • Line ending: CRLF (\r\n)
  • RFC 4180 compliant

Options

The following can be configured in Export Options:

  • Header Row — Whether to output column names in the first row
  • Type Info Row — Output column type information in the row after the header
  • Date Format — Output format for DATE/TIMESTAMP columns

Other Export Formats

Excel Export

Exports table data to an Excel file (.xlsx / .xls).

  1. Select a table and click the Excel button on the toolbar.
  2. Specify the destination file path.
  3. An Excel file is generated.

Access Export

Exports table data to a Microsoft Access database (.accdb / .mdb).

  1. Select a table and click the Access button on the toolbar.
  2. Specify the destination file path.
  3. A table is created in the Access database and data is imported.

SQL Server Export

Exports table data directly to a SQL Server database.

  1. Select a table and click the SQL Server button on the toolbar.
  2. Enter SQL Server connection information in the connection dialog.
  3. A table is created in the target database and data is inserted.

If a table with the same name already exists, it is automatically dropped and recreated.

ODBC Export

Exports table data to any ODBC-compatible database.

  1. Select a table and click the ODBC button on the toolbar.
  2. Enter the ODBC connection string or DSN in the connection dialog.
  3. A table is created in the target database and data is inserted.

If a table with the same name already exists, it is automatically dropped and recreated.
