Export Options

How to Access

Click the Options button on the toolbar to open the Export Options dialog.

Date Format

FormatExampleDescription
YYYY/MM/DD HH:MI:SS2026/01/15 09:30:00Standard format (default)
YYYYMMDD20260115Compact (date only)
YYYYMMDDHH24MISS20260115093000Compact (date and time)
CustomAnyTokens: YYYY, MM, DD, HH24, MI, SS

CSV Options

SettingDefaultDescription
Output Header RowONOutput column names in the first row
Output Type Info RowOFFOutput column types in the row after the header

SQL Options

A dialog is displayed before SQL export. The following options can be toggled via checkboxes:

SettingDefaultDescription
Output DROP TABLE + CREATE TABLEONOutput DROP TABLE IF EXISTS and CREATE TABLE DDL before INSERT statements. Toggleable via checkbox. Drops and recreates existing tables
DBMS TypeOracleOracle / PostgreSQL / MySQL / SQL Server
Note: Option settings are saved in application settings and persist across restarts.
