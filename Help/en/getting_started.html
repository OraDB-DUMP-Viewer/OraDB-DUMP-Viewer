Getting Started

What is OraDB DUMP Viewer?

OraDB DUMP Viewer is a standalone tool that lets you browse and export Oracle database dump files (EXP / EXPDP) without an Oracle client.

Key Features

  • Dump File Analysis — Supports both EXP (legacy) and EXPDP (Data Pump) formats
  • Table List — Displays schemas, table names, and column information in a tree view
  • Data Preview — Paginated table data display with sorting and copy support
  • Advanced Search — Filter data using 12 operators with AND/OR compound conditions
  • CSV Export — Export table data to CSV files
  • SQL Export — Generate INSERT statements (Oracle / PostgreSQL / MySQL / SQL Server)
  • Excel Export — Export to .xlsx / .xls format
  • Access Export — Export to .mdb / .accdb databases
  • SQL Server Export — Export directly to a SQL Server database
  • ODBC Export — Export to any ODBC-compatible database
  • Batch Export — Export multiple tables to CSV/SQL at once
  • LOB File Extraction — Extract BLOB/CLOB/NCLOB data to individual files
  • Workspace — Save and restore your working state
  • Table Exclusion — Hide unwanted tables (with undo/redo support)
  • Multilingual Support — 10 languages including Japanese, English, Chinese, Korean, German, French, Spanish, Italian, Russian, and Portuguese (BR)
  • Bug Reports — Report bugs directly to GitHub Issues from within the app
  • Auto Update — Detects new versions and provides one-click updates

Installation

  1. Run the distributed MSI installer.
  2. Follow the installation wizard instructions.
  3. Launch "OraDB DUMP Viewer" from the desktop or Start menu.
