LOB File Extraction

Overview

Extracts data stored in BLOB / CLOB / NCLOB columns of a table as one file per record.

Supported Data Types

Column TypeOutput Content
BLOBBinary files (images, PDF, ZIP, etc.)
CLOBText files (decoded using the dump file's character set)
NCLOBUnicode text files

Steps

  1. Select a table containing LOB columns in the table list.
  2. Select File(&F) > Extract Files(&F) from the menu.
  3. The LOB extraction dialog is displayed.
  4. Configure the following:
    • LOB Column — Select the LOB column to extract
    • Output Folder — Directory for the output files
    • File Naming — Sequential numbering or column value
    • Extension — File extension for output files (default: lob)
  5. Click the Extract button.

File Naming Methods

MethodExampleDescription
Sequential000001.jpg, 000002.jpg6-digit sequential numbers starting from 1
Column Valuefilename_col_value.jpgUses the value of a specified column as the file name

Usage Examples

Extracting Image Files

When JPEG images are stored in a BLOB column:

  1. LOB Column: Select IMAGE_DATA (BLOB)
  2. Extension: jpg
  3. File Naming: Sequential or ID column value

Extracting Text Data

When XML or JSON is stored in a CLOB column:

  1. LOB Column: Select XML_CONTENT (CLOB)
  2. Extension: xml
Caution: Extraction may take time when there are many LOB records. You can cancel via the progress dialog.
Note: In the table preview, LOB data is displayed as placeholders: %BLOB%, %CLOB%, %NCLOB%. Use this LOB File Extraction feature to retrieve the actual data.
