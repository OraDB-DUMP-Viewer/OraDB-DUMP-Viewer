Opening a Dump File

Steps

  1. Select File(&F) > Open Dump File(&O)... from the menu.
    Or click the Open button on the toolbar.
  2. In the file dialog, select a dump file (.dmp).
  3. Analysis begins and a progress bar is displayed.
  4. When complete, the table list appears in the workspace.

Auto-Detection

The dump file format (EXP / EXPDP / EXPDP Compressed) is detected automatically by analyzing the leading bytes and marker bytes of the file.

FormatDetection Method
EXPLeading bytes 0x01-0x05 + "EXPORT:" string
EXPDP"xml version" string
EXPDP (Compressed)"KGC" + "HDR" markers

Character Set

The dump file character set is also detected automatically. Supported character sets:

  • AL32UTF8 / UTF8
  • JA16SJIS (Shift_JIS)
  • JA16EUC (EUC-JP)
  • AL16UTF16
  • US7ASCII
  • WE8ISO8859P1 / WE8MSWIN1252
Note: For compressed EXPDP dumps, only table definitions (DDL) can be read. Decompression of data records is not supported.
