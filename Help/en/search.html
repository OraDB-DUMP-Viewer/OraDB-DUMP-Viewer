Search & Filter

Table Name Filter

Type part of a table name in the search text box at the top of the workspace to filter the table list in real time.

  • Case-insensitive.
  • Partial matching on table names.
  • Clear the text to remove the filter.

Advanced Search (Table Data Search)

On the table preview screen, you can perform advanced condition-based searches on data. Click the Advanced Search button to open the search condition dialog.

Setting Search Conditions

Specify the following for each condition row:

  1. Column Name — Select the target column
  2. Operator — Choose from 12 operators
  3. Search Value — Enter the comparison value (not required for Null / Not Null operators)
  4. Case Sensitive — Check to perform case-sensitive comparison

Compound Conditions (AND / OR)

When adding conditions, you can specify the logical operator between them as AND or OR.

  • AND — Show only rows that satisfy all conditions
  • OR — Show rows that satisfy any of the conditions

Condition rows can be freely added or removed, enabling complex filtering.

Preserving Search Conditions

Search conditions are automatically restored when reopening the same table. To clear conditions, use the Clear Search button on the preview screen.

Invert Selection

You can invert the selection state of the table list. Use Invert Selection from the context menu.
