Limitations

  • Data records in compressed EXPDP dumps cannot be read (metadata only)
  • Actual data for BFILE type cannot be extracted (path information only)
  • Encrypted dump files are not supported
  • Dumps via network link are not supported
  • Maximum number of tables: 1,000
  • Maximum number of columns: 1,000 (per table)

Character Sets

The following NLS character sets are supported:

  • AL32UTF8 / UTF8
  • JA16SJIS
  • JA16EUC
  • AL16UTF16
  • US7ASCII
  • WE8ISO8859P1 / WE8MSWIN1252
