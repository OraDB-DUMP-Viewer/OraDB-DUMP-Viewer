Table Preview

Opening a Table

Double-click a table name in the table list to open the data preview. Or right-click the table and select Open(&O).

Data Display

Data is displayed in a DataGridView.

  • Column names appear in the column headers.
  • Each row displays one record.
  • NULL values are shown as blank.
  • LOB data (BLOB/CLOB/NCLOB) is shown as %BLOB%, %CLOB%, %NCLOB%.

Paging

When there is a large amount of data, it is displayed page by page.

  • Rows per page — Adjustable via the numeric input at the top right (default: 100 rows)
  • Page info — Current page number and total pages are displayed
  • Use navigation buttons to switch between pages

Sorting

Click a column header to sort data by that column. Click again to toggle between ascending and descending order.

Copying Cells

Select cells in the preview and press Ctrl+C to copy to the clipboard. Multi-cell selection and copying is also supported.

Advanced Search

Click the Advanced Search button on the preview screen to filter data using 12 operators with AND/OR compound conditions. See Search & Filter for details.

Exporting from Preview

The following exports are available from the preview toolbar:

  • SQL — SQL script output
  • CSV — CSV file output
  • Excel — Excel file output
  • Access — Access database output
  • SQL Server — Direct output to SQL Server
  • ODBC — Output to an ODBC-compatible database

Table Properties

Right-click a table and select Properties(&P) to view column definitions (column name, data type, length, etc.).

Table Exclusion

You can temporarily hide unwanted tables.

  1. Select tables in the table list (multi-select supported).
  2. Choose Delete(&D) from the context menu, or press the Delete key.
  3. Excluded tables are shown in the status bar.

To undo exclusion:

  • Edit(&E) > Undo (Ctrl+Z)
  • Object(&O) > Clear All Filters(&F) to remove all exclusions
