Workspace

Overview

The workspace is a feature that manages dump file analysis results and working state together. You can save and restore table exclusion states and filter settings.

Saving a Workspace

  1. Select File(&F) > Save Workspace(&S) from the menu.
  2. On the first save, specify the file path (extension: .odvw).
  3. Subsequent saves will overwrite the existing file.

Save As

File(&F) > Save As(&A) lets you save the workspace under a different file name.

Opening a Workspace

  1. Select File(&F) > Workspace(&W)... from the menu.
  2. Select an .odvw file.
  3. The dump file is loaded automatically, and exclusion states and filters are restored.

Saved Information

ItemDescription
Dump File PathAbsolute path to the original dump file

SchemaCurrently selected schema
Expanded NodesTree view expansion state

Recent Files

Up to 5 recent workspaces and dump files are displayed at the bottom of the File menu. Click to quickly reopen them.

Tip: A save confirmation dialog appears before closing a workspace. If there are unsaved changes, select "Yes" to save them.
