Comprobar la versión

Seleccione Ayuda(&H) > Acerca de(&A) en el menú para comprobar la versión actual.

El cuadro de diálogo Acerca de compara automáticamente con la última versión de GitHub e indica si hay una actualización disponible.

  • Si usa la última versión: el número de versión se muestra en verde
  • Si hay una versión más reciente disponible: el número de versión se muestra en naranja

Actualización automática

Cuando hay una nueva versión disponible, aparece un botón de actualización en el cuadro de diálogo.

  1. Haga clic en el botón Actualizar a la última versión en el cuadro de diálogo Acerca de.
  2. Seleccione Sí en el cuadro de diálogo de confirmación.
  3. El último instalador MSI se descarga automáticamente (se muestra una barra de progreso).
  4. Después de la descarga, la aplicación se cierra y el instalador se inicia automáticamente.

También puede hacer clic en el enlace Abrir página de versiones para ver la página de versiones de GitHub en su navegador.

Informes de errores

Si encuentra un error o problema, repórtelo desde Ayuda(&H) > Informe de errores(&R).

La siguiente información se recopila automáticamente y se muestra de forma transparente en el cuadro de diálogo del informe:

  • Versión de la aplicación / versión de DLL
  • Versión del SO / versión de .NET
  • Arquitectura / resolución de pantalla / DPI
  • Información del archivo de volcado (solo tamaño y formato; las rutas no se envían)

Los informes se envían automáticamente como GitHub Issues a través de un Cloudflare Worker. No se recopila ni se envía información personal (nombre de usuario, nombre del PC, rutas de archivos, etc.).

Si el archivo de volcado es de 50 MB o menos, puede adjuntarlo opcionalmente al informe.

Soporte multilingüe

Se admiten 10 idiomas: japonés, inglés, chino (simplificado), coreano, alemán, francés, español, italiano, ruso y portugués (BR). El idioma se selecciona automáticamente según el idioma de visualización del sistema al iniciar.
