Exportación por lotes

Descripción general

Exporte múltiples tablas a archivos CSV o SQL de una sola vez. Cada tabla se genera como un archivo individual.

Pasos

  1. Haga clic en el botón Exportación por lotes de la barra de herramientas.
  2. Se muestra el cuadro de diálogo de exportación.
  3. Seleccione el formato de salida (CSV / SQL).
  4. Especifique la carpeta de salida.
  5. Confirme las tablas a exportar.
    Desmarque las tablas que no necesite.
  6. Haga clic en el botón Exportar.

Nombres de archivos de salida

Los nombres de archivo se generan automáticamente con el formato NombreEsquema.NombreTabla.csv o NombreEsquema.NombreTabla.sql.

Selección / exclusión de tablas

  • Botones Seleccionar todo / Deseleccionar todo para operaciones masivas
  • Invertir selección para alternar los estados de selección
  • Haga clic en casillas de verificación individuales para seleccionar tablas específicas
Consejo: Las tablas excluidas en el espacio de trabajo también se excluyen automáticamente de la exportación por lotes.

Salida de informes

Desde Objeto(&O) > Salida de informes(&R), también puede generar información de tablas en archivos de texto.

  • Lista de objetos — Genera una lista de texto con nombres de tablas y conteo de filas
  • Definiciones de tablas — Genera las definiciones de columnas (nombre, tipo, tamaño) de todas las tablas
