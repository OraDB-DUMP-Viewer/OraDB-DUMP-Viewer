Exportación CSV

Pasos

  1. Seleccione la tabla que desea exportar en la lista de tablas.
  2. Haga clic en el botón CSV de la barra de herramientas, o seleccione Objeto(&O) > Exportar > Datos(&D).
  3. Especifique la ruta del archivo de destino.
  4. La exportación se ejecuta y se muestra un cuadro de diálogo de progreso.

Formato de salida

  • Codificación: UTF-8 (con BOM)
  • Delimitador: Coma (,)
  • Comillas: Comillas dobles (") — se aplican cuando los valores contienen comas, saltos de línea o comillas dobles
  • Fin de línea: CRLF (\r\n)
  • Conforme a RFC 4180

Opciones

Lo siguiente se puede configurar en las Opciones de exportación:

  • Fila de encabezado — Si se incluyen los nombres de columna en la primera fila
  • Fila de información de tipos — Incluir información del tipo de columna en la fila posterior al encabezado
  • Formato de fecha — Formato de salida para columnas DATE/TIMESTAMP

Otros formatos de exportación

Exportación a Excel

Exporta datos de tablas a un archivo Excel (.xlsx / .xls).

  1. Seleccione una tabla y haga clic en el botón Excel de la barra de herramientas.
  2. Especifique la ruta del archivo de destino.
  3. Se genera un archivo Excel.

Exportación a Access

Exporta datos de tablas a una base de datos Microsoft Access (.accdb / .mdb).

  1. Seleccione una tabla y haga clic en el botón Access de la barra de herramientas.
  2. Especifique la ruta del archivo de destino.
  3. Se crea una tabla en la base de datos Access y se importan los datos.

Exportación a SQL Server

Exporta datos de tablas directamente a una base de datos SQL Server.

  1. Seleccione una tabla y haga clic en el botón SQL Server de la barra de herramientas.
  2. Introduzca la información de conexión de SQL Server en el cuadro de diálogo de conexión.
  3. Se crea una tabla en la base de datos de destino y se insertan los datos.

Si ya existe una tabla con el mismo nombre, se elimina y se recrea automáticamente.

Exportación ODBC

Exporta datos de tablas a cualquier base de datos compatible con ODBC.

  1. Seleccione una tabla y haga clic en el botón ODBC de la barra de herramientas.
  2. Introduzca la cadena de conexión ODBC o DSN en el cuadro de diálogo de conexión.
  3. Se crea una tabla en la base de datos de destino y se insertan los datos.

Si ya existe una tabla con el mismo nombre, se elimina y se recrea automáticamente.

Manejo de tablas existentes

Destino de exportaciónComportamiento con tablas existentes
Script SQLDROP TABLE IF EXISTS (activable mediante casilla de verificación)
SQL ServerEliminación automática + recreación
ODBCEliminación automática + recreación

Nota: Para las exportaciones a SQL Server y ODBC, la base de datos de destino debe crearse con antelación.
