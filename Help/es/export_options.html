Opciones de exportación

Cómo acceder

Haga clic en el botón Opciones de la barra de herramientas para abrir el cuadro de diálogo de opciones de exportación.

Opciones SQL

Se muestra un cuadro de diálogo antes de la exportación SQL. Las siguientes opciones se pueden activar mediante casillas de verificación:

ConfiguraciónPredeterminadoDescripción
Incluir DROP TABLE + CREATE TABLEONIncluir DROP TABLE IF EXISTS y CREATE TABLE DDL antes de las sentencias INSERT. Activable mediante casilla de verificación. Elimina y recrea tablas existentes
Tipo de DBMSOracleOracle / PostgreSQL / MySQL / SQL Server
Nota: La configuración de opciones se guarda en los ajustes de la aplicación y persiste entre reinicios.
