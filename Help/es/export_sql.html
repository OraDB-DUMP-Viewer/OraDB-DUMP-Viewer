Exportación SQL

Pasos

  1. Seleccione la tabla que desea exportar en la lista de tablas.
  2. Haga clic en el botón SQL de la barra de herramientas, o seleccione Objeto(&O) > Exportar > Script(&S).
  3. Especifique la ruta del archivo de destino.
  4. La exportación se ejecuta.

DBMS compatibles

Se pueden generar sentencias INSERT para los siguientes DBMS:

DBMSDiferencias de sintaxis
OracleFunción TO_DATE, tipo NUMBER tal cual
PostgreSQLSintaxis CAST, tipo TIMESTAMP
MySQLIdentificadores con acento grave, tipo DATETIME
SQL ServerIdentificadores con corchetes, tipo DATETIME2

Opciones

Se muestra un cuadro de diálogo antes de la exportación al hacer clic en el botón SQL. Se pueden configurar las siguientes opciones:

  • Tipo de DBMS — Elija entre Oracle / PostgreSQL / MySQL / SQL Server
  • Salida de DROP TABLE + CREATE TABLE (predeterminado: ON) — Activable mediante casilla de verificación. Cuando está activado, genera DROP TABLE IF EXISTS y CREATE TABLE DDL antes de las sentencias INSERT. Elimina y recrea la tabla si ya existe en la base de datos de destino.
  • Formato de fecha — Formato literal o llamada a función

Sintaxis de DROP TABLE por DBMS

DBMSSintaxis
Oracle / PostgreSQLDROP TABLE IF EXISTS "table_name" CASCADE;
MySQLDROP TABLE IF EXISTS `table_name`;
SQL ServerIF OBJECT_ID('table_name', 'U') IS NOT NULL DROP TABLE [table_name];
Nota: Los datos LOB (BLOB/CLOB) no se incluyen en las exportaciones SQL. Use la Extracción de archivos LOB para exportar datos LOB.
