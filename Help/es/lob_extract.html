Extracción de archivos LOB

Descripción general

Extrae los datos almacenados en columnas BLOB / CLOB / NCLOB de una tabla como un archivo por registro.

Tipos de datos compatibles

Tipo de columnaContenido de salida
BLOBArchivos binarios (imágenes, PDF, ZIP, etc.)
CLOBArchivos de texto (decodificados usando el juego de caracteres del archivo de volcado)
NCLOBArchivos de texto Unicode

Pasos

  1. Seleccione una tabla que contenga columnas LOB en la lista de tablas.
  2. Seleccione Archivo(&F) > Extraer archivos(&F) en el menú.
  3. Se muestra el cuadro de diálogo de extracción de LOB.
  4. Configure lo siguiente:
    • Columna LOB — Seleccione la columna LOB a extraer
    • Carpeta de salida — Directorio para los archivos de salida
    • Nomenclatura de archivos — Numeración secuencial o valor de columna
    • Extensión — Extensión de archivo para los archivos de salida (predeterminado: lob)
  5. Haga clic en el botón Extraer.

Métodos de nomenclatura de archivos

MétodoEjemploDescripción
Secuencial000001.jpg, 000002.jpgNúmeros secuenciales de 6 dígitos a partir de 1

Ejemplos de uso

Extraer archivos de imagen

Cuando hay imágenes JPEG almacenadas en una columna BLOB:

  1. Columna LOB: Seleccione IMAGE_DATA (BLOB)
  2. Extensión: jpg
  3. Nomenclatura de archivos: Secuencial o valor de columna ID

Extraer datos de texto

Cuando hay XML o JSON almacenado en una columna CLOB:

  1. Columna LOB: Seleccione XML_CONTENT (CLOB)
  2. Extensión: xml
Precaución: La extracción puede tardar cuando hay muchos registros LOB. Puede cancelar mediante el cuadro de diálogo de progreso.
Nota: En la vista previa de la tabla, los datos LOB se muestran como marcadores: %BLOB%, %CLOB%, %NCLOB%. Use esta función de extracción de archivos LOB para obtener los datos reales.
