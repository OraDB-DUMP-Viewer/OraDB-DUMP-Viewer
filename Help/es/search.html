Búsqueda y filtrado

Filtro por nombre de tabla

Escriba parte de un nombre de tabla en el cuadro de búsqueda en la parte superior del espacio de trabajo para filtrar la lista de tablas en tiempo real.

  • No distingue entre mayúsculas y minúsculas.
  • Coincidencia parcial en nombres de tablas.
  • Borre el texto para eliminar el filtro.

Búsqueda avanzada (búsqueda de datos de tabla)

En la pantalla de vista previa de la tabla, puede realizar búsquedas avanzadas basadas en condiciones sobre los datos. Haga clic en el botón Búsqueda avanzada para abrir el cuadro de diálogo de condiciones de búsqueda.

Configuración de condiciones de búsqueda

Especifique lo siguiente para cada fila de condición:

  1. Nombre de columna — Seleccione la columna objetivo
  2. Operador — Elija entre 12 operadores
  3. Valor de búsqueda — Introduzca el valor de comparación (no necesario para los operadores Null / Not Null)
  4. Distinguir mayúsculas — Marque para realizar una comparación que distinga mayúsculas de minúsculas

Condiciones compuestas (AND / OR)

Al agregar condiciones, puede especificar el operador lógico entre ellas como AND u OR.

  • AND — Muestra solo las filas que cumplen todas las condiciones
  • OR — Muestra las filas que cumplen cualquiera de las condiciones

Las filas de condiciones se pueden agregar o eliminar libremente, lo que permite un filtrado complejo.

Conservación de las condiciones de búsqueda

Las condiciones de búsqueda se restauran automáticamente al reabrir la misma tabla. Para borrar las condiciones, use el botón Borrar búsqueda en la pantalla de vista previa.

Invertir selección

Puede invertir el estado de selección de la lista de tablas. Use Invertir selección en el menú contextual.
