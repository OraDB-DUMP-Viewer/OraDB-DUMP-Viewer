Vista previa de tablas

Abrir una tabla

Haga doble clic en un nombre de tabla en la lista de tablas para abrir la vista previa de datos. O haga clic derecho en la tabla y seleccione Abrir(&O).

Visualización de datos

Los datos se muestran en un DataGridView.

  • Los nombres de columnas aparecen en los encabezados de columna.
  • Cada fila muestra un registro.
  • Los valores NULL se muestran en blanco.
  • Los datos LOB (BLOB/CLOB/NCLOB) se muestran como %BLOB%, %CLOB%, %NCLOB%.

Paginación

Cuando hay una gran cantidad de datos, se muestran página por página.

  • Filas por página — Ajustable mediante el campo numérico en la parte superior derecha (predeterminado: 100 filas)
  • Información de página — Se muestra el número de página actual y el total de páginas
  • Use los botones de navegación para cambiar entre páginas

Ordenación

Haga clic en un encabezado de columna para ordenar los datos por esa columna. Haga clic de nuevo para alternar entre orden ascendente y descendente.

Copiar celdas

Seleccione celdas en la vista previa y presione Ctrl+C para copiar al portapapeles. También se admite la selección y copia de múltiples celdas.

Búsqueda avanzada

Haga clic en el botón Búsqueda avanzada en la pantalla de vista previa para filtrar datos utilizando 12 operadores con condiciones compuestas AND/OR. Consulte Búsqueda y filtrado para más detalles.

Exportar desde la vista previa

Las siguientes exportaciones están disponibles desde la barra de herramientas de la vista previa:

  • SQL — Salida de script SQL
  • CSV — Salida de archivo CSV
  • Excel — Salida de archivo Excel
  • Access — Salida a base de datos Access
  • SQL Server — Salida directa a SQL Server
  • ODBC — Salida a base de datos compatible con ODBC

Propiedades de la tabla

Haga clic derecho en una tabla y seleccione Propiedades(&P) para ver las definiciones de columnas (nombre de columna, tipo de dato, longitud, etc.).

Exclusión de tablas

Puede ocultar temporalmente las tablas no deseadas.

  1. Seleccione tablas en la lista de tablas (se admite selección múltiple).
  2. Elija Eliminar(&D) en el menú contextual, o presione la tecla Delete.
  3. Las tablas excluidas se muestran en la barra de estado.

Para deshacer la exclusión:

  • Editar(&E) > Deshacer (Ctrl+Z)
  • Objeto(&O) > Borrar todos los filtros(&F) para eliminar todas las exclusiones
