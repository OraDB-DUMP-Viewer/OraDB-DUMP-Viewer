CSV エクスポート

操作手順

  1. テーブル一覧でエクスポートしたいテーブルを選択します。
  2. ツールバーの CSV ボタンをクリック、 または オブジェクト(&O) > エクスポート > データ(&D) を選択します。
  3. 保存先のファイルパスを指定します。
  4. エクスポートが実行され、進捗ダイアログが表示されます。

出力形式

  • 文字コード: UTF-8 (BOM 付き)
  • 区切り: カンマ (,)
  • 引用符: ダブルクオート (") — 値にカンマ・改行・ダブルクオートが含まれる場合に適用
  • 改行: CRLF (\r\n)
  • RFC 4180 準拠

オプション

エクスポートオプション で以下を設定できます:

  • ヘッダー行 — 1行目にカラム名を出力するかどうか
  • 型情報行 — ヘッダーの次行にカラムの型情報を出力
  • 日付書式 — DATE/TIMESTAMP カラムの出力書式

その他のエクスポート形式

Excel エクスポート

テーブルデータを Excel ファイル (.xlsx / .xls) に出力します。

  1. テーブルを選択し、ツールバーの Excel ボタンをクリックします。
  2. 保存先のファイルパスを指定します。
  3. Excel ファイルが生成されます。

Access エクスポート

テーブルデータを Microsoft Access データベース (.accdb / .mdb) に出力します。

  1. テーブルを選択し、ツールバーの Access ボタンをクリックします。
  2. 保存先のファイルパスを指定します。
  3. Access データベースにテーブルが作成され、データがインポートされます。

SQL Server エクスポート

テーブルデータを SQL Server データベースに直接出力します。

  1. テーブルを選択し、ツールバーの SQL Server ボタンをクリックします。
  2. 接続ダイアログで SQL Server への接続情報を入力します。
  3. 対象データベースにテーブルが作成され、データが挿入されます。

ODBC エクスポート

テーブルデータを任意の ODBC 対応データベースに出力します。

  1. テーブルを選択し、ツールバーの ODBC ボタンをクリックします。
  2. 接続ダイアログで ODBC 接続文字列または DSN を入力します。
  3. 対象データベースにテーブルが作成され、データが挿入されます。
注意: SQL Server エクスポートおよび ODBC エクスポートでは、 接続先のデータベースが事前に作成されている必要があります。
