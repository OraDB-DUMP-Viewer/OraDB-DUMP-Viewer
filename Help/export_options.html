エクスポートオプション

アクセス方法

ツールバーの オプション ボタンをクリックすると、 エクスポートオプションダイアログが表示されます。

日付書式

書式例説明
YYYY/MM/DD HH:MI:SS2026/01/15 09:30:00標準形式 (デフォルト)
YYYYMMDD20260115コンパクト (日付のみ)
YYYYMMDDHH24MISS20260115093000コンパクト (日時)
カスタム任意トークン: YYYY, MM, DD, HH24, MI, SS

CSV オプション

設定デフォルト説明
ヘッダー行を出力ON1行目にカラム名を出力
型情報行を出力OFFヘッダーの次行にカラム型を出力

SQL オプション

設定デフォルト説明
CREATE TABLE を出力OFFINSERT 文の前に DDL を出力
DBMS 種別OracleOracle / PostgreSQL / MySQL / SQL Server
注意: オプション設定はアプリケーション終了時にリセットされます。
