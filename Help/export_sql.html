SQL エクスポート

操作手順

  1. テーブル一覧でエクスポートしたいテーブルを選択します。
  2. ツールバーの SQL ボタンをクリック、 または オブジェクト(&O) > エクスポート > スクリプト(&S) を選択します。
  3. 保存先のファイルパスを指定します。
  4. エクスポートが実行されます。

対応 DBMS

INSERT 文は以下の DBMS 向けに生成できます:

DBMS構文の違い
OracleTO_DATE 関数, NUMBER 型そのまま
PostgreSQLCAST 構文, TIMESTAMP 型
MySQLバッククオート識別子, DATETIME 型
SQL Serverブラケット識別子, DATETIME2 型

オプション

  • CREATE TABLE 出力 — INSERT 文の前に CREATE TABLE DDL を出力
  • 日付書式 — リテラル形式またはファンクション呼び出し
注意: LOB データ (BLOB/CLOB) は SQL エクスポートに含まれません。 LOB データをエクスポートするには LOB ファイル抽出 を使用してください。
