Vérification de la version

Sélectionnez Aide(&H) > À propos(&A) dans le menu pour vérifier la version actuelle.

La boîte de dialogue À propos compare automatiquement avec la dernière version GitHub et indique si une mise à jour est disponible.

  • Si vous utilisez la dernière version : le numéro de version est affiché en vert
  • Si une version plus récente est disponible : le numéro de version est affiché en orange

Mise à jour automatique

Lorsqu'une nouvelle version est disponible, un bouton de mise à jour apparaît dans la boîte de dialogue.

  1. Cliquez sur le bouton Mettre à jour vers la dernière version dans la boîte de dialogue À propos.
  2. Sélectionnez Oui dans la boîte de dialogue de confirmation.
  3. Le dernier programme d'installation MSI est téléchargé automatiquement (une barre de progression s'affiche).
  4. Après le téléchargement, l'application se ferme et le programme d'installation se lance automatiquement.

Vous pouvez également cliquer sur le lien Ouvrir la page des versions pour consulter la page des versions GitHub dans votre navigateur.

Signalement de bugs

Si vous trouvez un bug ou un problème, signalez-le depuis Aide(&H) > Signaler un bug(&R).

Les informations suivantes sont automatiquement collectées et affichées de manière transparente dans la boîte de dialogue de signalement :

  • Version de l'application / Version des DLL
  • Version du système d'exploitation / Version de .NET
  • Architecture / Résolution d'écran / DPI
  • Informations du fichier dump (taille et format uniquement ; les chemins ne sont pas envoyés)

Les signalements sont automatiquement soumis en tant que GitHub Issues via un Cloudflare Worker. Aucune information personnelle (nom d'utilisateur, nom du PC, chemins de fichiers, etc.) n'est collectée ni envoyée.

Si le fichier dump fait 50 Mo ou moins, vous pouvez optionnellement le joindre au signalement.

Support multilingue

10 langues sont prises en charge : japonais, anglais, chinois (simplifié), coréen, allemand, français, espagnol, italien, russe et portugais (BR). La langue est automatiquement sélectionnée en fonction de la langue d'affichage du système au démarrage.
