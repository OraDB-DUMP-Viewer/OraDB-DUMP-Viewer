Export par lots

Présentation

Exportez plusieurs tables vers des fichiers CSV ou SQL en une seule opération. Chaque table est exportée dans un fichier individuel.

Étapes

  1. Cliquez sur le bouton Export par lots de la barre d'outils.
  2. La boîte de dialogue d'export s'affiche.
  3. Sélectionnez le format de sortie (CSV / SQL).
  4. Spécifiez le dossier de sortie.
  5. Confirmez les tables à exporter.
    Décochez les tables dont vous n'avez pas besoin.
  6. Cliquez sur le bouton Exporter.

Noms des fichiers de sortie

Les noms de fichiers sont générés automatiquement au format NomSchéma.NomTable.csv ou NomSchéma.NomTable.sql.

Sélection / Exclusion de tables

  • Boutons Tout sélectionner / Tout désélectionner pour les opérations en masse
  • Inverser la sélection pour basculer les états de sélection
  • Cliquez sur les cases à cocher individuelles pour sélectionner des tables spécifiques
Astuce : Les tables exclues dans l'espace de travail sont automatiquement exclues de l'export par lots également.

Sortie de rapport

Depuis Objet(&O) > Sortie de rapport(&R), vous pouvez également exporter les informations de tables vers des fichiers texte.

  • Liste des objets — Exporter une liste textuelle des noms de tables et du nombre de lignes
  • Définitions de tables — Exporter les définitions de colonnes (nom, type, taille) pour toutes les tables
