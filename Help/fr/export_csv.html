Export CSV

Étapes

  1. Sélectionnez la table que vous souhaitez exporter dans la liste des tables.
  2. Cliquez sur le bouton CSV de la barre d'outils, ou sélectionnez Objet(&O) > Exporter > Données(&D).
  3. Spécifiez le chemin du fichier de destination.
  4. L'export s'exécute et une boîte de dialogue de progression s'affiche.

Format de sortie

  • Encodage : UTF-8 (avec BOM)
  • Délimiteur : Virgule (,)
  • Guillemets : Guillemets doubles (") — appliqués lorsque les valeurs contiennent des virgules, des retours à la ligne ou des guillemets doubles
  • Fin de ligne : CRLF (\r\n)
  • Conforme à la RFC 4180

Options

Les éléments suivants peuvent être configurés dans les Options d'export :

  • Ligne d'en-tête — Inclure ou non les noms de colonnes dans la première ligne
  • Ligne d'informations de type — Inclure les informations de type de colonnes dans la ligne suivant l'en-tête
  • Format de date — Format de sortie pour les colonnes DATE/TIMESTAMP

Autres formats d'export

Export Excel

Exporte les données de table vers un fichier Excel (.xlsx / .xls).

  1. Sélectionnez une table et cliquez sur le bouton Excel de la barre d'outils.
  2. Spécifiez le chemin du fichier de destination.
  3. Un fichier Excel est généré.

Export Access

Exporte les données de table vers une base de données Microsoft Access (.accdb / .mdb).

  1. Sélectionnez une table et cliquez sur le bouton Access de la barre d'outils.
  2. Spécifiez le chemin du fichier de destination.
  3. Une table est créée dans la base de données Access et les données sont importées.

Export SQL Server

Exporte les données de table directement vers une base de données SQL Server.

  1. Sélectionnez une table et cliquez sur le bouton SQL Server de la barre d'outils.
  2. Saisissez les informations de connexion SQL Server dans la boîte de dialogue de connexion.
  3. Une table est créée dans la base de données cible et les données sont insérées.

Si une table du même nom existe déjà, elle est automatiquement supprimée et recréée.

Export ODBC

Exporte les données de table vers toute base de données compatible ODBC.

  1. Sélectionnez une table et cliquez sur le bouton ODBC de la barre d'outils.
  2. Saisissez la chaîne de connexion ODBC ou le DSN dans la boîte de dialogue de connexion.
  3. Une table est créée dans la base de données cible et les données sont insérées.

Si une table du même nom existe déjà, elle est automatiquement supprimée et recréée.
