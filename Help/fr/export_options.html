Options d'export

Comment y accéder

Cliquez sur le bouton Options de la barre d'outils pour ouvrir la boîte de dialogue des options d'export.

Options SQL

Une boîte de dialogue s'affiche avant l'export SQL. Les options suivantes peuvent être activées/désactivées via des cases à cocher :

ParamètrePar défautDescription
Sortie DROP TABLE + CREATE TABLEACTIVÉGénérer les DDL DROP TABLE IF EXISTS et CREATE TABLE avant les instructions INSERT. Activable/désactivable via case à cocher. Supprime et recrée les tables existantes
Type de SGBDOracleOracle / PostgreSQL / MySQL / SQL Server
Remarque : Les paramètres d'options sont enregistrés dans les paramètres de l'application et persistent après redémarrage.
