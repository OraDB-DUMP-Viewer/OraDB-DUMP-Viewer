Qu'est-ce qu'OraDB DUMP Viewer ?

OraDB DUMP Viewer est un outil autonome qui vous permet de parcourir et d'exporter des fichiers dump de bases de données Oracle (EXP / EXPDP) sans client Oracle.

Fonctionnalités principales

  • Analyse de fichiers dump — Prend en charge les formats EXP (traditionnel) et EXPDP (Data Pump)
  • Liste des tables — Affiche les schémas, les noms de tables et les informations de colonnes dans une arborescence
  • Aperçu des données — Affichage paginé des données de table avec tri et copie
  • Recherche avancée — Filtrage des données avec 12 opérateurs et conditions composées AND/OR
  • Export CSV — Exporter les données de table vers des fichiers CSV
  • Export SQL — Générer des instructions INSERT (Oracle / PostgreSQL / MySQL / SQL Server)
  • Export Excel — Exporter au format .xlsx / .xls
  • Export Access — Exporter vers des bases de données .mdb / .accdb
  • Export SQL Server — Exporter directement vers une base de données SQL Server
  • Export ODBC — Exporter vers toute base de données compatible ODBC
  • Export par lots — Exporter plusieurs tables en CSV/SQL en une seule opération
  • Extraction de fichiers LOB — Extraire les données BLOB/CLOB/NCLOB en fichiers individuels
  • Espace de travail — Sauvegarder et restaurer votre état de travail
  • Exclusion de tables — Masquer les tables non souhaitées (avec annulation/rétablissement)
  • Support multilingue — 10 langues dont le japonais, l'anglais, le chinois, le coréen, l'allemand, le français, l'espagnol, l'italien, le russe et le portugais (BR)
  • Signalement de bugs — Signaler des bugs directement sur GitHub Issues depuis l'application
  • Mise à jour automatique — Détecte les nouvelles versions et propose une mise à jour en un clic

EXPDP compresséexpdp COMPRESSION=ALLPartiel (définitions de tables uniquement)

Installation

  1. Exécutez le programme d'installation MSI fourni.
  2. Suivez les instructions de l'assistant d'installation.
  3. Lancez « OraDB DUMP Viewer » depuis le bureau ou le menu Démarrer.
