Extraction de fichiers LOB

Présentation

Extrait les données stockées dans les colonnes BLOB / CLOB / NCLOB d'une table en un fichier par enregistrement.

Types de données pris en charge

Type de colonneContenu de sortie
BLOBFichiers binaires (images, PDF, ZIP, etc.)
CLOBFichiers texte (décodés avec le jeu de caractères du fichier dump)
NCLOBFichiers texte Unicode

Étapes

  1. Sélectionnez une table contenant des colonnes LOB dans la liste des tables.
  2. Sélectionnez Fichier(&F) > Extraire les fichiers(&F) dans le menu.
  3. La boîte de dialogue d'extraction LOB s'affiche.
  4. Configurez les éléments suivants :
    • Colonne LOB — Sélectionnez la colonne LOB à extraire
    • Dossier de sortie — Répertoire pour les fichiers de sortie
    • Nommage des fichiers — Numérotation séquentielle ou valeur de colonne
    • Extension — Extension des fichiers de sortie (par défaut : lob)
  5. Cliquez sur le bouton Extraire.

Méthodes de nommage des fichiers

MéthodeExempleDescription
Séquentiel000001.jpg, 000002.jpgNuméros séquentiels à 6 chiffres à partir de 1
Valeur de colonnevaleur_col_nom.jpgUtilise la valeur d'une colonne spécifiée comme nom de fichier

Exemples d'utilisation

Extraction de fichiers image

Lorsque des images JPEG sont stockées dans une colonne BLOB :

  1. Colonne LOB : Sélectionnez IMAGE_DATA (BLOB)
  2. Extension : jpg
  3. Nommage : Séquentiel ou valeur de la colonne ID

Extraction de données texte

Lorsque du XML ou du JSON est stocké dans une colonne CLOB :

  1. Colonne LOB : Sélectionnez XML_CONTENT (CLOB)
  2. Extension : xml
Attention : L'extraction peut prendre du temps lorsqu'il y a de nombreux enregistrements LOB. Vous pouvez annuler via la boîte de dialogue de progression.
Remarque : Dans l'aperçu de table, les données LOB sont affichées sous forme d'espaces réservés : %BLOB%, %CLOB%, %NCLOB%. Utilisez cette fonctionnalité d'extraction de fichiers LOB pour récupérer les données réelles.
