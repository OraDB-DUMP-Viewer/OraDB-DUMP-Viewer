Ouvrir un fichier dump

Étapes

  1. Sélectionnez Fichier(&F) > Ouvrir un fichier dump(&O)... dans le menu.
    Ou cliquez sur le bouton Ouvrir de la barre d'outils.
  2. Dans la boîte de dialogue, sélectionnez un fichier dump (.dmp).
  3. L'analyse démarre et une barre de progression s'affiche.
  4. Une fois terminé, la liste des tables apparaît dans l'espace de travail.

Détection automatique

Le format du fichier dump (EXP / EXPDP / EXPDP compressé) est détecté automatiquement en analysant les octets d'en-tête et les octets marqueurs du fichier.

FormatMéthode de détection
EXPOctets d'en-tête 0x01-0x05 + chaîne "EXPORT:"
EXPDPChaîne "xml version"
EXPDP (Compressé)Marqueurs "KGC" + "HDR"

Jeu de caractères

Le jeu de caractères du fichier dump est également détecté automatiquement. Jeux de caractères pris en charge :

  • AL32UTF8 / UTF8
  • JA16SJIS (Shift_JIS)
  • JA16EUC (EUC-JP)
  • AL16UTF16
  • US7ASCII
  • WE8ISO8859P1 / WE8MSWIN1252
Remarque : Pour les dumps EXPDP compressés, seules les définitions de tables (DDL) peuvent être lues. La décompression des enregistrements de données n'est pas prise en charge.
