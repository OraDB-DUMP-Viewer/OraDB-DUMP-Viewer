Aperçu des tables

Ouvrir une table

Double-cliquez sur un nom de table dans la liste des tables pour ouvrir l'aperçu des données. Vous pouvez aussi faire un clic droit sur la table et sélectionner Ouvrir(&O).

Affichage des données

Les données sont affichées dans un DataGridView.

  • Les noms de colonnes apparaissent dans les en-têtes de colonnes.
  • Chaque ligne affiche un enregistrement.
  • Les valeurs NULL sont affichées comme vides.
  • Les données LOB (BLOB/CLOB/NCLOB) sont affichées sous forme de %BLOB%, %CLOB%, %NCLOB%.

Pagination

Lorsqu'il y a un grand volume de données, celles-ci sont affichées page par page.

  • Lignes par page — Ajustable via le champ numérique en haut à droite (par défaut : 100 lignes)
  • Info de page — Le numéro de page actuel et le nombre total de pages sont affichés
  • Utilisez les boutons de navigation pour changer de page

Tri

Cliquez sur un en-tête de colonne pour trier les données par cette colonne. Cliquez à nouveau pour alterner entre l'ordre croissant et décroissant.

Copie de cellules

Sélectionnez des cellules dans l'aperçu et appuyez sur Ctrl+C pour copier dans le presse-papiers. La sélection et la copie multi-cellules sont également prises en charge.

Recherche avancée

Cliquez sur le bouton Recherche avancée sur l'écran d'aperçu pour filtrer les données avec 12 opérateurs et des conditions composées AND/OR. Consultez Recherche et filtrage pour plus de détails.

Exporter depuis l'aperçu

Les exports suivants sont disponibles depuis la barre d'outils de l'aperçu :

  • SQL — Sortie de script SQL
  • CSV — Sortie de fichier CSV
  • Excel — Sortie de fichier Excel
  • Access — Sortie vers une base de données Access
  • SQL Server — Sortie directe vers SQL Server
  • ODBC — Sortie vers une base de données compatible ODBC

Propriétés de la table

Faites un clic droit sur une table et sélectionnez Propriétés(&P) pour afficher les définitions de colonnes (nom de colonne, type de données, longueur, etc.).

Exclusion de tables

Vous pouvez masquer temporairement les tables non souhaitées.

  1. Sélectionnez des tables dans la liste des tables (sélection multiple possible).
  2. Choisissez Supprimer(&D) dans le menu contextuel, ou appuyez sur la touche Suppr.
  3. Les tables exclues sont affichées dans la barre d'état.

Pour annuler l'exclusion :

  • Édition(&E) > Annuler (Ctrl+Z)
  • Objet(&O) > Effacer tous les filtres(&F) pour supprimer toutes les exclusions
