はじめに

OraDB DUMP Viewer とは

OraDB DUMP Viewer は、Oracle データベースのダンプファイル (EXP / EXPDP) を Oracle クライアントなしで閲覧・エクスポートできるスタンドアロンツールです。

主な機能

  • ダンプファイル解析 — EXP (従来形式) と EXPDP (Data Pump) 両方に対応
  • テーブル一覧表示 — スキーマ・テーブル名・カラム情報をツリー表示
  • データプレビュー — テーブルデータを DataGridView で閲覧
  • CSV エクスポート — テーブルデータを CSV ファイルに出力
  • SQL エクスポート — INSERT 文を生成 (Oracle / PostgreSQL / MySQL / SQL Server)
  • 一括エクスポート — 複数テーブルを一括で CSV/SQL 出力
  • LOB ファイル抽出 — BLOB/CLOB/NCLOB データを個別ファイルに抽出
  • ワークスペース — 作業状態の保存・復元
  • テーブル除外 — 不要テーブルの非表示 (元に戻す対応)

対応ダンプ形式

形式ツール対応状況
EXP (従来形式)exp / imp対応
EXPDP (Data Pump)expdp / impdp対応
EXPDP 圧縮expdp COMPRESSION=ALL対応 (メタデータのみ)

インストール

  1. 配布された MSI インストーラーを実行します。
  2. インストールウィザードの指示に従ってインストールします。
  3. デスクトップまたはスタートメニューから「OraDB DUMP Viewer」を起動します。
