はじめに

OraDB DUMP Viewer とは

OraDB DUMP Viewer は、Oracle データベースのダンプファイル (EXP / EXPDP) を Oracle クライアントなしで閲覧・エクスポートできるスタンドアロンツールです。

主な機能

  • ダンプファイル解析 — EXP (従来形式) と EXPDP (Data Pump) 両方に対応
  • テーブル一覧表示 — スキーマ・テーブル名・カラム情報をツリー表示
  • データプレビュー — テーブルデータをページング表示 (ソート・コピー対応)
  • 高度な検索 — 12種類の演算子、AND/OR 複合条件でデータをフィルタ
  • CSV エクスポート — テーブルデータを CSV ファイルに出力
  • SQL エクスポート — INSERT 文を生成 (Oracle / PostgreSQL / MySQL / SQL Server)
  • Excel エクスポート — .xlsx / .xls 形式で出力
  • Access エクスポート — .mdb / .accdb データベースに出力
  • SQL Server エクスポート — SQL Server データベースに直接接続して出力
  • ODBC エクスポート — 任意の ODBC 対応データベースに出力
  • 一括エクスポート — 複数テーブルを一括で CSV/SQL 出力
  • LOB ファイル抽出 — BLOB/CLOB/NCLOB データを個別ファイルに抽出
  • ワークスペース — 作業状態の保存・復元
  • テーブル除外 — 不要テーブルの非表示 (元に戻す/やり直し対応)
  • 多言語対応 — 日本語・英語・中国語(簡体)・韓国語・ドイツ語・フランス語・スペイン語・イタリア語・ロシア語・ポルトガル語(BR) の10言語に対応
  • エラー報告 — アプリ内からバグを直接 GitHub Issues に報告
  • 自動更新 — 新バージョンの検出とワンクリック更新

対応ダンプ形式

形式ツール対応状況
EXP (従来形式)exp / imp対応 (Oracle 7〜11g)
EXPDP (Data Pump)expdp / impdp対応 (Oracle 10g〜23ai)
EXPDP 圧縮expdp COMPRESSION=ALL一部対応 (テーブル定義のみ)

インストール

  1. 配布された MSI インストーラーを実行します。
  2. インストールウィザードの指示に従ってインストールします。
  3. デスクトップまたはスタートメニューから「OraDB DUMP Viewer」を起動します。
