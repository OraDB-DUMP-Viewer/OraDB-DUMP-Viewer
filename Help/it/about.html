Verifica della versione

Selezionare Guida(&H) > Informazioni(&A) dal menu per verificare la versione corrente.

La finestra di dialogo Informazioni confronta automaticamente con l'ultima release su GitHub e indica se è disponibile un aggiornamento.

  • Se si utilizza l'ultima versione: il numero di versione viene visualizzato in verde
  • Se è disponibile una versione più recente: il numero di versione viene visualizzato in arancione

Aggiornamento automatico

Quando è disponibile una nuova versione, nella finestra di dialogo appare un pulsante di aggiornamento.

  1. Fare clic sul pulsante Aggiorna all'ultima versione nella finestra di dialogo Informazioni.
  2. Selezionare Sì nella finestra di conferma.
  3. L'ultimo programma di installazione MSI viene scaricato automaticamente (viene visualizzata una barra di avanzamento).
  4. Dopo il download, l'applicazione si chiude e il programma di installazione si avvia automaticamente.

È inoltre possibile fare clic sul collegamento Apri pagina release per visualizzare la pagina delle release su GitHub nel browser.

Segnalazione bug

Se si riscontra un bug o un problema, segnalarlo da Guida(&H) > Segnalazione bug(&R).

Le seguenti informazioni vengono raccolte automaticamente e visualizzate in modo trasparente nella finestra di dialogo del report:

  • Versione dell'applicazione / Versione DLL
  • Versione del sistema operativo / Versione .NET
  • Architettura / Risoluzione dello schermo / DPI
  • Informazioni sul file dump (solo dimensione e formato; i percorsi non vengono inviati)

I report vengono inviati automaticamente come GitHub Issues tramite un Cloudflare Worker. Nessuna informazione personale (nome utente, nome PC, percorsi dei file, ecc.) viene raccolta o inviata.

Se il file dump è di 50 MB o inferiore, è possibile allegarlo facoltativamente al report.

Supporto multilingue

Sono supportate 10 lingue: giapponese, inglese, cinese (semplificato), coreano, tedesco, francese, spagnolo, italiano, russo e portoghese (BR). La lingua viene selezionata automaticamente in base alla lingua di visualizzazione del sistema all'avvio.
