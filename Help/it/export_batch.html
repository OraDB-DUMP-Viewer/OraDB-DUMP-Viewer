Esportazione batch

Panoramica

Esporta più tabelle in file CSV o SQL contemporaneamente. Ogni tabella viene generata come file individuale.

Procedura

  1. Fare clic sul pulsante Esportazione batch nella barra degli strumenti.
  2. Viene visualizzata la finestra di dialogo di esportazione.
  3. Selezionare il formato di output (CSV / SQL).
  4. Specificare la cartella di output.
  5. Confermare le tabelle da esportare.
    Deselezionare le tabelle non necessarie.
  6. Fare clic sul pulsante Esporta.

Nomi dei file di output

I nomi dei file vengono generati automaticamente nel formato NomeSchema.NomeTabella.csv o NomeSchema.NomeTabella.sql.

Selezione / Esclusione tabelle

  • Pulsanti Seleziona tutto / Deseleziona tutto per operazioni in blocco
  • Inverti selezione per invertire gli stati di selezione
  • Fare clic sulle singole caselle di controllo per selezionare tabelle specifiche
Suggerimento: Le tabelle escluse nell'area di lavoro vengono automaticamente escluse anche dall'esportazione batch.

Output report

Da Oggetto(&O) > Output report(&R), è possibile generare informazioni sulle tabelle in file di testo.

  • Elenco oggetti — Genera un elenco testuale dei nomi delle tabelle e del numero di righe
  • Definizioni tabelle — Genera le definizioni delle colonne (nome, tipo, dimensione) per tutte le tabelle
