Esportazione CSV

Procedura

  1. Selezionare la tabella da esportare nell'elenco delle tabelle.
  2. Fare clic sul pulsante CSV nella barra degli strumenti, oppure selezionare Oggetto(&O) > Esporta > Dati(&D).
  3. Specificare il percorso del file di destinazione.
  4. L'esportazione viene eseguita e viene visualizzata una finestra di avanzamento.

Formato di output

  • Codifica: UTF-8 (con BOM)
  • Delimitatore: Virgola (,)
  • Virgolette: Doppie virgolette (") — applicate quando i valori contengono virgole, ritorni a capo o virgolette doppie
  • Fine riga: CRLF (\r\n)
  • Conforme a RFC 4180

Opzioni

Le seguenti opzioni possono essere configurate nelle Opzioni di esportazione:

  • Riga di intestazione — Se includere i nomi delle colonne nella prima riga
  • Riga informazioni tipo — Includere le informazioni sul tipo di colonna nella riga successiva all'intestazione
  • Formato data — Formato di output per le colonne DATE/TIMESTAMP

Altri formati di esportazione

Esportazione Excel

Esporta i dati della tabella in un file Excel (.xlsx / .xls).

  1. Selezionare una tabella e fare clic sul pulsante Excel nella barra degli strumenti.
  2. Specificare il percorso del file di destinazione.
  3. Viene generato un file Excel.

Esportazione Access

Esporta i dati della tabella in un database Microsoft Access (.accdb / .mdb).

  1. Selezionare una tabella e fare clic sul pulsante Access nella barra degli strumenti.
  2. Specificare il percorso del file di destinazione.
  3. Viene creata una tabella nel database Access e i dati vengono importati.

Esportazione SQL Server

Esporta i dati della tabella direttamente in un database SQL Server.

  1. Selezionare una tabella e fare clic sul pulsante SQL Server nella barra degli strumenti.
  2. Inserire le informazioni di connessione a SQL Server nella finestra di dialogo di connessione.
  3. Viene creata una tabella nel database di destinazione e i dati vengono inseriti.

Se esiste già una tabella con lo stesso nome, viene automaticamente eliminata e ricreata.

Esportazione ODBC

Esporta i dati della tabella in qualsiasi database compatibile ODBC.

  1. Selezionare una tabella e fare clic sul pulsante ODBC nella barra degli strumenti.
  2. Inserire la stringa di connessione ODBC o il DSN nella finestra di dialogo di connessione.
  3. Viene creata una tabella nel database di destinazione e i dati vengono inseriti.

Se esiste già una tabella con lo stesso nome, viene automaticamente eliminata e ricreata.
