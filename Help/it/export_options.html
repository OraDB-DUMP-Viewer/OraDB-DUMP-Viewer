Opzioni di esportazione

Come accedere

Fare clic sul pulsante Opzioni nella barra degli strumenti per aprire la finestra di dialogo Opzioni di esportazione.

Opzioni SQL

Una finestra di dialogo viene visualizzata prima dell'esportazione SQL. Le seguenti opzioni possono essere attivate/disattivate tramite caselle di controllo:

ImpostazionePredefinitoDescrizione
Output DROP TABLE + CREATE TABLEONGenera DROP TABLE IF EXISTS e CREATE TABLE DDL prima delle istruzioni INSERT. Attivabile/disattivabile tramite casella di controllo. Elimina e ricrea le tabelle esistenti
Tipo DBMSOracleOracle / PostgreSQL / MySQL / SQL Server
Nota: Le impostazioni delle opzioni vengono salvate nelle impostazioni dell'applicazione e persistono tra i riavvii.
