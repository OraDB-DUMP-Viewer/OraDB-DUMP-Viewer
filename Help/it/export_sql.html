Esportazione SQL

Procedura

  1. Selezionare la tabella da esportare nell'elenco delle tabelle.
  2. Fare clic sul pulsante SQL nella barra degli strumenti, oppure selezionare Oggetto(&O) > Esporta > Script(&S).
  3. Specificare il percorso del file di destinazione.
  4. L'esportazione viene eseguita.

DBMS supportati

Le istruzioni INSERT possono essere generate per i seguenti DBMS:

DBMSDifferenze di sintassi
OracleFunzione TO_DATE, tipo NUMBER invariato
PostgreSQLSintassi CAST, tipo TIMESTAMP
MySQLIdentificatori con backtick, tipo DATETIME
SQL ServerIdentificatori tra parentesi quadre, tipo DATETIME2

Opzioni

Una finestra di dialogo viene visualizzata prima dell'esportazione quando si fa clic sul pulsante SQL. Le seguenti opzioni possono essere configurate:

  • Tipo DBMS — Scegliere tra Oracle / PostgreSQL / MySQL / SQL Server
  • Output DROP TABLE + CREATE TABLE (predefinito: ON) — Attivabile/disattivabile tramite casella di controllo. Quando attivo, genera DROP TABLE IF EXISTS e CREATE TABLE DDL prima delle istruzioni INSERT. Elimina e ricrea la tabella se esiste già nel database di destinazione.
  • Formato data — Formato letterale o chiamata a funzione

Sintassi DROP TABLE per DBMS

DBMSSintassi
Oracle / PostgreSQLDROP TABLE IF EXISTS "table_name" CASCADE;
MySQLDROP TABLE IF EXISTS `table_name`;
SQL ServerIF OBJECT_ID('table_name', 'U') IS NOT NULL DROP TABLE [table_name];
Nota: I dati LOB (BLOB/CLOB) non sono inclusi nelle esportazioni SQL. Utilizzare l'Estrazione file LOB per esportare i dati LOB.
