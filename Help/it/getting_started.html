Introduzione

Che cos'è OraDB DUMP Viewer?

OraDB DUMP Viewer è uno strumento autonomo che consente di esplorare ed esportare file dump di database Oracle (EXP / EXPDP) senza un client Oracle.

Funzionalità principali

  • Analisi file dump — Supporta sia il formato EXP (legacy) che EXPDP (Data Pump)
  • Elenco tabelle — Visualizza schemi, nomi delle tabelle e informazioni sulle colonne in una vista ad albero
  • Anteprima dati — Visualizzazione paginata dei dati della tabella con ordinamento e supporto alla copia
  • Ricerca avanzata — Filtraggio dei dati con 12 operatori e condizioni composte AND/OR
  • Esportazione CSV — Esportazione dei dati della tabella in file CSV
  • Esportazione SQL — Generazione di istruzioni INSERT (Oracle / PostgreSQL / MySQL / SQL Server)
  • Esportazione Excel — Esportazione in formato .xlsx / .xls
  • Esportazione Access — Esportazione in database .mdb / .accdb
  • Esportazione SQL Server — Esportazione diretta in un database SQL Server
  • Esportazione ODBC — Esportazione in qualsiasi database compatibile ODBC
  • Esportazione batch — Esportazione di più tabelle in CSV/SQL contemporaneamente
  • Estrazione file LOB — Estrazione dei dati BLOB/CLOB/NCLOB in file individuali
  • Area di lavoro — Salvataggio e ripristino dello stato di lavoro
  • Esclusione tabelle — Nasconde le tabelle indesiderate (con supporto annulla/ripristina)
  • Supporto multilingue — 10 lingue tra cui giapponese, inglese, cinese, coreano, tedesco, francese, spagnolo, italiano, russo e portoghese (BR)
  • Segnalazione bug — Segnalazione dei bug direttamente a GitHub Issues dall'interno dell'applicazione
  • Aggiornamento automatico — Rileva nuove versioni e offre aggiornamenti con un clic

Formati dump supportati

FormatoStrumentoStato del supporto

EXPDP compressoexpdp COMPRESSION=ALLParziale (solo definizioni tabelle)

Installazione

  1. Eseguire il programma di installazione MSI distribuito.
  2. Seguire le istruzioni della procedura guidata di installazione.
  3. Avviare "OraDB DUMP Viewer" dal desktop o dal menu Start.
