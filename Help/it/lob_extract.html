Estrazione file LOB

Panoramica

Estrae i dati memorizzati nelle colonne BLOB / CLOB / NCLOB di una tabella come un file per ogni record.

Tipi di dati supportati

Tipo di colonnaContenuto dell'output
BLOBFile binari (immagini, PDF, ZIP, ecc.)
CLOBFile di testo (decodificati utilizzando il set di caratteri del file dump)
NCLOBFile di testo Unicode

Procedura

  1. Selezionare una tabella contenente colonne LOB nell'elenco delle tabelle.
  2. Selezionare File(&F) > Estrai file(&F) dal menu.
  3. Viene visualizzata la finestra di dialogo di estrazione LOB.
  4. Configurare i seguenti parametri:
    • Colonna LOB — Selezionare la colonna LOB da estrarre
    • Cartella di output — Directory per i file di output
    • Denominazione file — Numerazione sequenziale o valore della colonna
    • Estensione — Estensione dei file di output (predefinita: lob)
  5. Fare clic sul pulsante Estrai.

Metodi di denominazione dei file

MetodoEsempioDescrizione
Sequenziale000001.jpg, 000002.jpgNumeri sequenziali a 6 cifre a partire da 1
Valore colonnavalore_colonna_nome.jpgUtilizza il valore di una colonna specificata come nome del file

Esempi di utilizzo

Estrazione di file immagine

Quando immagini JPEG sono memorizzate in una colonna BLOB:

  1. Colonna LOB: Selezionare IMAGE_DATA (BLOB)
  2. Estensione: jpg
  3. Denominazione file: Sequenziale o valore colonna ID

Estrazione di dati testuali

Quando XML o JSON sono memorizzati in una colonna CLOB:

  1. Colonna LOB: Selezionare XML_CONTENT (CLOB)
  2. Estensione: xml
Attenzione: L'estrazione potrebbe richiedere tempo quando sono presenti molti record LOB. È possibile annullare tramite la finestra di avanzamento.
Nota: Nell'anteprima della tabella, i dati LOB vengono visualizzati come segnaposto: %BLOB%, %CLOB%, %NCLOB%. Utilizzare questa funzionalità di estrazione file LOB per recuperare i dati effettivi.
