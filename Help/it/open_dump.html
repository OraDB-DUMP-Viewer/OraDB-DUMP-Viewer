Apertura di un file dump

Procedura

  1. Selezionare File(&F) > Apri file dump(&O)... dal menu.
    Oppure fare clic sul pulsante Apri nella barra degli strumenti.
  2. Nella finestra di dialogo, selezionare un file dump (.dmp).
  3. L'analisi inizia e viene visualizzata una barra di avanzamento.
  4. Al termine, l'elenco delle tabelle appare nell'area di lavoro.

Rilevamento automatico

Il formato del file dump (EXP / EXPDP / EXPDP compresso) viene rilevato automaticamente analizzando i byte iniziali e i byte marcatori del file.

FormatoMetodo di rilevamento
EXPByte iniziali 0x01-0x05 + stringa "EXPORT:"
EXPDPStringa "xml version"
EXPDP (compresso)Marcatori "KGC" + "HDR"

Set di caratteri

Anche il set di caratteri del file dump viene rilevato automaticamente. Set di caratteri supportati:

  • AL32UTF8 / UTF8
  • JA16SJIS (Shift_JIS)
  • JA16EUC (EUC-JP)
  • AL16UTF16
  • US7ASCII
  • WE8ISO8859P1 / WE8MSWIN1252
Nota: Per i dump EXPDP compressi, è possibile leggere solo le definizioni delle tabelle (DDL). La decompressione dei record di dati non è supportata.
