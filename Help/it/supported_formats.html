Limitazioni

  • I record di dati nei dump EXPDP compressi non possono essere letti (solo metadati)
  • I dati effettivi del tipo BFILE non possono essere estratti (solo informazioni sul percorso)
  • I file dump crittografati non sono supportati
  • I dump tramite network link non sono supportati
  • Numero massimo di tabelle: 1.000
  • Numero massimo di colonne: 1.000 (per tabella)

Set di caratteri

Sono supportati i seguenti set di caratteri NLS:

  • AL32UTF8 / UTF8
  • JA16SJIS
  • JA16EUC
  • AL16UTF16
  • US7ASCII
  • WE8ISO8859P1 / WE8MSWIN1252
