Anteprima tabella

Apertura di una tabella

Fare doppio clic sul nome di una tabella nell'elenco delle tabelle per aprire l'anteprima dei dati. Oppure fare clic con il tasto destro sulla tabella e selezionare Apri(&O).

Visualizzazione dei dati

I dati vengono visualizzati in un DataGridView.

  • I nomi delle colonne appaiono nelle intestazioni delle colonne.
  • Ogni riga visualizza un record.
  • I valori NULL vengono mostrati come vuoti.
  • I dati LOB (BLOB/CLOB/NCLOB) vengono mostrati come %BLOB%, %CLOB%, %NCLOB%.

Paginazione

Quando la quantità di dati è elevata, vengono visualizzati pagina per pagina.

  • Righe per pagina — Regolabile tramite il campo numerico in alto a destra (predefinito: 100 righe)
  • Info pagina — Vengono visualizzati il numero di pagina corrente e il totale delle pagine
  • Utilizzare i pulsanti di navigazione per passare da una pagina all'altra

Ordinamento

Fare clic sull'intestazione di una colonna per ordinare i dati in base a quella colonna. Fare clic di nuovo per alternare tra ordine crescente e decrescente.

Copia delle celle

Selezionare le celle nell'anteprima e premere Ctrl+C per copiare negli appunti. È supportata anche la selezione e la copia di più celle.

Ricerca avanzata

Fare clic sul pulsante Ricerca avanzata nella schermata di anteprima per filtrare i dati utilizzando 12 operatori con condizioni composte AND/OR. Consultare Ricerca e filtro per i dettagli.

Esportazione dall'anteprima

Le seguenti esportazioni sono disponibili dalla barra degli strumenti dell'anteprima:

  • SQL — Output script SQL
  • CSV — Output file CSV
  • Excel — Output file Excel
  • Access — Output database Access
  • SQL Server — Output diretto in SQL Server
  • ODBC — Output in un database compatibile ODBC

Proprietà della tabella

Fare clic con il tasto destro su una tabella e selezionare Proprietà(&P) per visualizzare le definizioni delle colonne (nome colonna, tipo di dato, lunghezza, ecc.).

Esclusione tabelle

È possibile nascondere temporaneamente le tabelle indesiderate.

  1. Selezionare le tabelle nell'elenco delle tabelle (selezione multipla supportata).
  2. Scegliere Elimina(&D) dal menu contestuale, oppure premere il tasto Canc.
  3. Le tabelle escluse vengono mostrate nella barra di stato.

Per annullare l'esclusione:

  • Modifica(&E) > Annulla (Ctrl+Z)
  • Oggetto(&O) > Cancella tutti i filtri(&F) per rimuovere tutte le esclusioni
