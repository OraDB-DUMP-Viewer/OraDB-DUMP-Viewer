Guida di OraDB DUMP Viewer

OraDB DUMP Viewer è uno strumento autonomo per esplorare ed esportare file dump Oracle Data Pump (EXPDP) e nel formato legacy EXP tramite un'interfaccia grafica, senza necessità di un client Oracle.
