Area di lavoro

Panoramica

L'area di lavoro è una funzionalità che gestisce insieme i risultati dell'analisi dei file dump e lo stato di lavoro. È possibile salvare e ripristinare gli stati di esclusione delle tabelle e le impostazioni dei filtri.

Salvataggio di un'area di lavoro

  1. Selezionare File(&F) > Salva area di lavoro(&S) dal menu.
  2. Al primo salvataggio, specificare il percorso del file (estensione: .odvw).
  3. I salvataggi successivi sovrascriveranno il file esistente.

Salva con nome

File(&F) > Salva con nome(&A) consente di salvare l'area di lavoro con un nome file diverso.

Apertura di un'area di lavoro

  1. Selezionare File(&F) > Area di lavoro(&W)... dal menu.
  2. Selezionare un file .odvw.
  3. Il file dump viene caricato automaticamente e gli stati di esclusione e i filtri vengono ripristinati.

Informazioni salvate

ElementoDescrizione
Percorso file dumpPercorso assoluto del file dump originale
Tabelle escluseElenco delle tabelle nascoste
Filtro di ricercaTesto di ricerca del nome tabella
SchemaSchema attualmente selezionato
Nodi espansiStato di espansione della vista ad albero

File recenti

Fino a 5 aree di lavoro e file dump recenti vengono visualizzati in fondo al menu File. Fare clic per riaprirli rapidamente.

Suggerimento: Una finestra di dialogo di conferma salvataggio appare prima della chiusura di un'area di lavoro. Se ci sono modifiche non salvate, selezionare "Sì" per salvarle.
