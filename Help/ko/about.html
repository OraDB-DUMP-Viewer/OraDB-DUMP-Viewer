버전 확인

메뉴에서 도움말(&H) > 정보(&A)를 선택하여 현재 버전을 확인할 수 있습니다.

정보 대화 상자는 최신 GitHub 릴리스와 자동으로 비교하여 업데이트 가능 여부를 표시합니다.

  • 최신 버전 사용 중: 버전 번호가 녹색으로 표시됩니다
  • 새 버전이 있는 경우: 버전 번호가 주황색으로 표시됩니다

자동 업데이트

새 버전을 사용할 수 있는 경우 대화 상자에 업데이트 버튼이 표시됩니다.

  1. 정보 대화 상자에서 최신 버전으로 업데이트 버튼을 클릭합니다.
  2. 확인 대화 상자에서 예를 선택합니다.
  3. 최신 MSI 설치 프로그램이 자동으로 다운로드됩니다 (진행 표시줄이 표시됩니다).
  4. 다운로드 완료 후 애플리케이션이 종료되고 설치 프로그램이 자동으로 실행됩니다.

릴리스 페이지 열기 링크를 클릭하여 브라우저에서 GitHub 릴리스 페이지를 확인할 수도 있습니다.

라이선스 인증

처음 실행 시 라이선스 인증이 필요합니다. https://www.odv.dev/에서 라이선스를 취득하세요.

라이선스 검증에는 로컬 RSA 서명 검증과 시작 시 온라인 유효성 검증이 포함됩니다. 오프라인 환경에서는 로컬 검증만으로 사용할 수 있습니다.

재인증 또는 라이선스 키 변경은 도움말(&H) > 라이선스 인증에서 수행할 수 있습니다.

버그 보고

버그나 문제를 발견한 경우 도움말(&H) > 버그 보고(&R)에서 보고할 수 있습니다.

보고 대화 상자에서 다음 정보가 자동으로 수집되어 투명하게 표시됩니다:

  • 애플리케이션 버전 / DLL 버전
  • OS 버전 / .NET 버전
  • 아키텍처 / 화면 해상도 / DPI
  • 덤프 파일 정보 (크기 및 형식만; 경로는 전송되지 않음)

보고서는 Cloudflare Worker를 통해 GitHub Issues로 자동 제출됩니다. 개인 정보 (사용자 이름, PC 이름, 파일 경로 등)는 수집되거나 전송되지 않습니다.

덤프 파일이 50MB 이하인 경우 보고서에 첨부할 수 있습니다.

다국어 지원

일본어, 영어, 중국어(간체), 한국어, 독일어, 프랑스어, 스페인어, 이탈리아어, 러시아어, 포르투갈어(BR) 등 10개 언어를 지원합니다. 시작 시 시스템 표시 언어에 따라 언어가 자동으로 선택됩니다.
