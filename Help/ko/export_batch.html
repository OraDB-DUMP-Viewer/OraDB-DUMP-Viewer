일괄 내보내기

개요

여러 테이블을 CSV 또는 SQL 파일로 한 번에 내보냅니다. 각 테이블은 개별 파일로 출력됩니다.

절차

  1. 도구 모음의 일괄 내보내기 버튼을 클릭합니다.
  2. 내보내기 대화 상자가 표시됩니다.
  3. 출력 형식(CSV / SQL)을 선택합니다.
  4. 출력 폴더를 지정합니다.
  5. 내보낼 테이블을 확인합니다.
    필요 없는 테이블은 체크를 해제합니다.
  6. 내보내기 버튼을 클릭합니다.

출력 파일 이름

파일 이름은 SchemaName.TableName.csv 또는 SchemaName.TableName.sql 형식으로 자동 생성됩니다.

테이블 선택 / 제외

  • 전체 선택 / 전체 해제 버튼으로 일괄 조작
  • 선택 반전으로 선택 상태 전환
  • 개별 체크박스를 클릭하여 특정 테이블 선택
팁: 워크스페이스에서 제외된 테이블은 일괄 내보내기에서도 자동으로 제외됩니다.

보고서 출력

개체(&O) > 보고서 출력(&R)에서 테이블 정보를 텍스트 파일로 출력할 수도 있습니다.

  • 오브젝트 목록 — 테이블 이름과 행 수의 텍스트 목록 출력
  • 테이블 정의 — 모든 테이블의 컬럼 정의(이름, 유형, 크기) 출력
