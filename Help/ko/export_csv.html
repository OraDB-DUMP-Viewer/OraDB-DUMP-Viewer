CSV 내보내기

절차

  1. 테이블 목록에서 내보낼 테이블을 선택합니다.
  2. 도구 모음의 CSV 버튼을 클릭하거나 개체(&O) > 내보내기 > 데이터(&D)를 선택합니다.
  3. 대상 파일 경로를 지정합니다.
  4. 내보내기가 실행되고 진행률 대화 상자가 표시됩니다.

출력 형식

  • 인코딩: UTF-8 (BOM 포함)
  • 구분자: 쉼표 (,)
  • 인용 부호: 큰따옴표 (") — 값에 쉼표, 줄 바꿈 또는 큰따옴표가 포함된 경우 적용
  • 줄 끝: CRLF (\r\n)
  • RFC 4180 준수

옵션

내보내기 옵션에서 다음을 설정할 수 있습니다:

  • 헤더 행 — 첫 번째 행에 컬럼 이름을 출력할지 여부
  • 유형 정보 행 — 헤더 다음 행에 컬럼 유형 정보를 출력
  • 날짜 형식 — DATE/TIMESTAMP 컬럼의 출력 형식

기타 내보내기 형식

Excel 내보내기

테이블 데이터를 Excel 파일(.xlsx / .xls)로 내보냅니다.

  1. 테이블을 선택하고 도구 모음의 Excel 버튼을 클릭합니다.
  2. 대상 파일 경로를 지정합니다.
  3. Excel 파일이 생성됩니다.

Access 내보내기

테이블 데이터를 Microsoft Access 데이터베이스(.accdb / .mdb)로 내보냅니다.

  1. 테이블을 선택하고 도구 모음의 Access 버튼을 클릭합니다.
  2. 대상 파일 경로를 지정합니다.
  3. Access 데이터베이스에 테이블이 생성되고 데이터가 가져오기됩니다.

SQL Server 내보내기

테이블 데이터를 SQL Server 데이터베이스로 직접 내보냅니다.

  1. 테이블을 선택하고 도구 모음의 SQL Server 버튼을 클릭합니다.
  2. 연결 대화 상자에서 SQL Server 연결 정보를 입력합니다.
  3. 대상 데이터베이스에 테이블이 생성되고 데이터가 삽입됩니다.

동일한 이름의 테이블이 이미 존재하는 경우 자동으로 삭제 후 다시 생성됩니다.

ODBC 내보내기

테이블 데이터를 ODBC 호환 데이터베이스로 내보냅니다.

  1. 테이블을 선택하고 도구 모음의 ODBC 버튼을 클릭합니다.
  2. 연결 대화 상자에서 ODBC 연결 문자열 또는 DSN을 입력합니다.
  3. 대상 데이터베이스에 테이블이 생성되고 데이터가 삽입됩니다.

동일한 이름의 테이블이 이미 존재하는 경우 자동으로 삭제 후 다시 생성됩니다.
