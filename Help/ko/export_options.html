형식예시설명
YYYY/MM/DD HH:MI:SS2026/01/15 09:30:00표준 형식 (기본값)
YYYYMMDD20260115간결한 형식 (날짜만)
YYYYMMDDHH24MISS20260115093000간결한 형식 (날짜와 시간)

설정기본값설명

SQL 옵션

SQL 내보내기 전에 대화 상자가 표시됩니다. 다음 옵션을 체크박스로 전환할 수 있습니다:

설정기본값설명
DROP TABLE + CREATE TABLE 출력ONINSERT 문 앞에 DROP TABLE IF EXISTS 및 CREATE TABLE DDL을 출력합니다. 체크박스로 전환 가능. 기존 테이블을 삭제 후 재생성합니다
DBMS 유형OracleOracle / PostgreSQL / MySQL / SQL Server
참고: 옵션 설정은 애플리케이션 설정에 저장되며 재시작 후에도 유지됩니다.
