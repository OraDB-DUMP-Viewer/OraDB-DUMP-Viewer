OraDB DUMP Viewer란?

OraDB DUMP Viewer는 Oracle 클라이언트 없이 Oracle 데이터베이스 덤프 파일(EXP / EXPDP)을 탐색하고 내보낼 수 있는 독립 실행형 도구입니다.

주요 기능

  • 덤프 파일 분석 — EXP(레거시) 및 EXPDP(Data Pump) 형식 모두 지원
  • 테이블 목록 — 스키마, 테이블 이름, 컬럼 정보를 트리 뷰로 표시
  • 데이터 미리보기 — 정렬 및 복사를 지원하는 페이지별 테이블 데이터 표시
  • 고급 검색 — 12개 연산자와 AND/OR 복합 조건으로 데이터 필터링
  • CSV 내보내기 — 테이블 데이터를 CSV 파일로 내보내기
  • SQL 내보내기 — INSERT 문 생성 (Oracle / PostgreSQL / MySQL / SQL Server)
  • Excel 내보내기 — .xlsx / .xls 형식으로 내보내기
  • Access 내보내기 — .mdb / .accdb 데이터베이스로 내보내기
  • SQL Server 내보내기 — SQL Server 데이터베이스로 직접 내보내기
  • ODBC 내보내기 — ODBC 호환 데이터베이스로 내보내기
  • 일괄 내보내기 — 여러 테이블을 CSV/SQL로 한 번에 내보내기
  • LOB 파일 추출 — BLOB/CLOB/NCLOB 데이터를 개별 파일로 추출
  • 워크스페이스 — 작업 상태 저장 및 복원
  • 테이블 제외 — 불필요한 테이블 숨기기 (실행 취소/다시 실행 지원)
  • 다국어 지원 — 일본어, 영어, 중국어, 한국어, 독일어, 프랑스어, 스페인어, 이탈리아어, 러시아어, 포르투갈어(BR) 등 10개 언어
  • 버그 신고 — 앱 내에서 직접 GitHub Issues로 버그 신고
  • 자동 업데이트 — 새 버전을 감지하고 원클릭 업데이트 제공

지원 덤프 형식

형식도구지원 상태
EXP (레거시)exp / imp지원 (Oracle 7–11g)
EXPDP (Data Pump)expdp / impdp지원 (Oracle 10g–23ai)
EXPDP 압축expdp COMPRESSION=ALL부분 지원 (테이블 정의만)

설치

  1. 배포된 MSI 설치 프로그램을 실행합니다.
  2. 설치 마법사의 지시에 따릅니다.
  3. 바탕 화면 또는 시작 메뉴에서 "OraDB DUMP Viewer"를 실행합니다.
