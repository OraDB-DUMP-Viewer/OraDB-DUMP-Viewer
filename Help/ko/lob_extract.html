LOB 파일 추출

개요

테이블의 BLOB / CLOB / NCLOB 컬럼에 저장된 데이터를 레코드당 하나의 파일로 추출합니다.

지원 데이터 타입

컬럼 타입출력 내용
BLOB바이너리 파일 (이미지, PDF, ZIP 등)
CLOB텍스트 파일 (덤프 파일의 문자 세트로 디코딩)
NCLOB유니코드 텍스트 파일

절차

  1. 테이블 목록에서 LOB 컬럼이 포함된 테이블을 선택합니다.
  2. 메뉴에서 파일(&F) > 파일 추출(&F)을 선택합니다.
  3. LOB 추출 대화 상자가 표시됩니다.
  4. 다음을 설정합니다:
    • LOB 컬럼 — 추출할 LOB 컬럼 선택
    • 출력 폴더 — 출력 파일의 디렉터리
    • 파일 명명 — 순번 또는 컬럼 값
    • 확장자 — 출력 파일의 확장자 (기본값: lob)
  5. 추출 버튼을 클릭합니다.

파일 명명 방법

방법예설명
순번000001.jpg, 000002.jpg1부터 시작하는 6자리 순번
컬럼 값filename_col_value.jpg지정 컬럼의 값을 파일 이름으로 사용

사용 예

이미지 파일 추출

BLOB 컬럼에 JPEG 이미지가 저장된 경우:

  1. LOB 컬럼: IMAGE_DATA (BLOB) 선택
  2. 확장자: jpg
  3. 파일 명명: 순번 또는 ID 컬럼 값

텍스트 데이터 추출

CLOB 컬럼에 XML 또는 JSON이 저장된 경우:

  1. LOB 컬럼: XML_CONTENT (CLOB) 선택
  2. 확장자: xml
주의: LOB 레코드가 많은 경우 추출에 시간이 걸릴 수 있습니다. 진행 대화 상자에서 취소할 수 있습니다.
참고: 테이블 미리보기에서 LOB 데이터는 %BLOB%, %CLOB%, %NCLOB%으로 표시됩니다. 실제 데이터를 가져오려면 이 LOB 파일 추출 기능을 사용하세요.
