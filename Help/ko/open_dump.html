덤프 파일 열기

절차

  1. 메뉴에서 파일(&F) > 덤프 파일 열기(&O)...를 선택합니다.
    또는 도구 모음의 열기 버튼을 클릭합니다.
  2. 파일 대화 상자에서 덤프 파일(.dmp)을 선택합니다.
  3. 분석이 시작되고 진행률 표시줄이 표시됩니다.
  4. 완료되면 워크스페이스에 테이블 목록이 표시됩니다.

자동 감지

덤프 파일 형식(EXP / EXPDP / EXPDP 압축)은 파일의 선행 바이트와 마커 바이트를 분석하여 자동으로 감지됩니다.

형식감지 방법
EXP선행 바이트 0x01-0x05 + "EXPORT:" 문자열
EXPDP"xml version" 문자열
EXPDP (압축)"KGC" + "HDR" 마커

문자 집합

덤프 파일의 문자 집합도 자동으로 감지됩니다. 지원되는 문자 집합:

  • AL32UTF8 / UTF8
  • JA16SJIS (Shift_JIS)
  • JA16EUC (EUC-JP)
  • AL16UTF16
  • US7ASCII
  • WE8ISO8859P1 / WE8MSWIN1252
참고: 압축된 EXPDP 덤프의 경우 테이블 정의(DDL)만 읽을 수 있습니다. 데이터 레코드의 압축 해제는 지원되지 않습니다.
