검색 & 필터

테이블 이름 필터

워크스페이스 상단의 검색 텍스트 상자에 테이블 이름의 일부를 입력하면 테이블 목록이 실시간으로 필터링됩니다.

  • 대소문자를 구분하지 않습니다.
  • 테이블 이름에 대한 부분 일치입니다.
  • 텍스트를 지우면 필터가 해제됩니다.

고급 검색 (테이블 데이터 검색)

테이블 미리보기 화면에서 데이터에 대해 조건 기반 고급 검색을 수행할 수 있습니다. 고급 검색 버튼을 클릭하면 검색 조건 대화 상자가 열립니다.

검색 조건 설정

각 조건 행에 다음을 지정합니다:

  1. 컬럼 이름 — 대상 컬럼 선택
  2. 연산자 — 12개 연산자 중 선택
  3. 검색 값 — 비교 값 입력 (Null / Not Null 연산자에는 필요 없음)
  4. 대소문자 구분 — 체크하면 대소문자를 구분하여 비교

연산자 목록

연산자설명예시
포함값에 검색 문자열이 포함됨NAME 포함 "Smith"
포함하지 않음값에 검색 문자열이 포함되지 않음NAME 포함하지 않음 "test"
같음값이 검색 문자열과 정확히 일치STATUS 같음 "ACTIVE"
같지 않음값이 검색 문자열과 일치하지 않음TYPE 같지 않음 "TEMP"
시작 문자값이 검색 문자열로 시작CODE 시작 문자 "A"
끝 문자값이 검색 문자열로 끝남EMAIL 끝 문자 ".com"
>값이 검색 값보다 큼AMOUNT > "1000"
<값이 검색 값보다 작음AGE < "30"
>=값이 검색 값 이상SCORE >= "80"
<=값이 검색 값 이하PRICE <= "500"
Null값이 NULL임MEMO Null
Not Null값이 NULL이 아님EMAIL Not Null

복합 조건 (AND / OR)

조건을 추가할 때 조건 간의 논리 연산자를 AND 또는 OR로 지정할 수 있습니다.

  • AND — 모든 조건을 만족하는 행만 표시
  • OR — 조건 중 하나라도 만족하는 행을 표시

조건 행은 자유롭게 추가하거나 제거할 수 있어 복잡한 필터링이 가능합니다.

검색 조건 유지

동일한 테이블을 다시 열 때 검색 조건이 자동으로 복원됩니다. 조건을 지우려면 미리보기 화면의 검색 지우기 버튼을 사용하세요.

선택 반전

테이블 목록의 선택 상태를 반전할 수 있습니다. 컨텍스트 메뉴에서 선택 반전을 사용하세요.
