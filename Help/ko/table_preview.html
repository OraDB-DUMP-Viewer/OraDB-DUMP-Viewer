테이블 미리보기

테이블 열기

테이블 목록에서 테이블 이름을 더블 클릭하면 데이터 미리보기가 열립니다. 또는 테이블을 마우스 오른쪽 버튼으로 클릭하고 열기(&O)를 선택합니다.

데이터 표시

데이터는 DataGridView에 표시됩니다.

  • 컬럼 이름이 컬럼 헤더에 표시됩니다.
  • 각 행은 하나의 레코드를 표시합니다.
  • NULL 값은 빈칸으로 표시됩니다.
  • LOB 데이터(BLOB/CLOB/NCLOB)는 %BLOB%, %CLOB%, %NCLOB%로 표시됩니다.

페이징

데이터가 많은 경우 페이지별로 표시됩니다.

  • 페이지당 행 수 — 오른쪽 상단의 숫자 입력으로 조정 가능 (기본값: 100행)
  • 페이지 정보 — 현재 페이지 번호와 전체 페이지 수가 표시됩니다
  • 탐색 버튼을 사용하여 페이지 간 이동

정렬

컬럼 헤더를 클릭하면 해당 컬럼으로 데이터를 정렬합니다. 다시 클릭하면 오름차순과 내림차순 사이를 전환합니다.

셀 복사

미리보기에서 셀을 선택하고 Ctrl+C를 누르면 클립보드에 복사됩니다. 다중 셀 선택 및 복사도 지원됩니다.

고급 검색

미리보기 화면에서 고급 검색 버튼을 클릭하면 12개 연산자와 AND/OR 복합 조건으로 데이터를 필터링할 수 있습니다. 자세한 내용은 검색 & 필터를 참조하세요.

미리보기에서 내보내기

미리보기 도구 모음에서 다음 내보내기를 사용할 수 있습니다:

  • SQL — SQL 스크립트 출력
  • CSV — CSV 파일 출력
  • Excel — Excel 파일 출력
  • Access — Access 데이터베이스 출력
  • SQL Server — SQL Server로 직접 출력
  • ODBC — ODBC 호환 데이터베이스로 출력

테이블 속성

테이블을 마우스 오른쪽 버튼으로 클릭하고 속성(&P)을 선택하면 컬럼 정의(컬럼 이름, 데이터 유형, 길이 등)를 확인할 수 있습니다.

테이블 제외

불필요한 테이블을 일시적으로 숨길 수 있습니다.

  1. 테이블 목록에서 테이블을 선택합니다 (다중 선택 지원).
  2. 컨텍스트 메뉴에서 삭제(&D)를 선택하거나 Delete 키를 누릅니다.
  3. 제외된 테이블은 상태 표시줄에 표시됩니다.

제외를 취소하려면:

  • 편집(&E) > 실행 취소 (Ctrl+Z)
  • 개체(&O) > 모든 필터 지우기(&F)로 모든 제외 해제
