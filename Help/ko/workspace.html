워크스페이스

개요

워크스페이스는 덤프 파일 분석 결과와 작업 상태를 함께 관리하는 기능입니다. 테이블 제외 상태와 필터 설정을 저장하고 복원할 수 있습니다.

워크스페이스 저장

  1. 메뉴에서 파일(&F) > 워크스페이스 저장(&S)을 선택합니다.
  2. 처음 저장할 때 파일 경로를 지정합니다 (확장자: .odvw).
  3. 이후 저장 시에는 기존 파일을 덮어씁니다.

다른 이름으로 저장

파일(&F) > 다른 이름으로 저장(&A)을 사용하면 워크스페이스를 다른 파일 이름으로 저장할 수 있습니다.

워크스페이스 열기

  1. 메뉴에서 파일(&F) > 워크스페이스(&W)...를 선택합니다.
  2. .odvw 파일을 선택합니다.
  3. 덤프 파일이 자동으로 로드되고 제외 상태와 필터가 복원됩니다.

저장되는 정보

항목설명
덤프 파일 경로원본 덤프 파일의 절대 경로
제외된 테이블숨겨진 테이블 목록
검색 필터테이블 이름 검색 텍스트
스키마현재 선택된 스키마
확장된 노드트리 뷰 확장 상태

최근 파일

파일 메뉴 하단에 최근 워크스페이스 및 덤프 파일이 최대 5개까지 표시됩니다. 클릭하면 빠르게 다시 열 수 있습니다.

팁: 워크스페이스를 닫기 전에 저장 확인 대화 상자가 표시됩니다. 저장하지 않은 변경 사항이 있으면 "예"를 선택하여 저장하세요.
