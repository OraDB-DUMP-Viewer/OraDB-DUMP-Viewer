LOB ファイル抽出

概要

テーブルの BLOB / CLOB / NCLOB カラムに格納されたデータを、 1レコードにつき1ファイルとして抽出します。

対応データ型

カラム型出力内容
BLOBバイナリファイル (画像、PDF、ZIP 等)
CLOBテキストファイル (ダンプの文字セットでデコード)
NCLOBUnicode テキストファイル

操作手順

  1. テーブル一覧で LOB カラムを含むテーブルを選択します。
  2. メニューの ファイル(&F) > ファイルの取り出し(&F) を選択します。
  3. LOB 抽出ダイアログが表示されます。
  4. 以下を設定します:
    • LOB カラム — 抽出する LOB カラムを選択
    • 出力フォルダ — ファイルの出力先ディレクトリ
    • ファイル名方式 — 連番 または カラム値
    • 拡張子 — 出力ファイルの拡張子 (デフォルト: lob)
  5. 抽出 ボタンをクリックします。

ファイル名方式

方式例説明
連番000001.jpg, 000002.jpg1 から始まる6桁連番
カラム値filename_col_value.jpg指定カラムの値をファイル名に使用

使用例

画像ファイルの抽出

BLOB カラムに JPEG 画像が格納されている場合:

  1. LOB カラム: IMAGE_DATA (BLOB) を選択
  2. 拡張子: jpg
  3. ファイル名方式: 連番 または ID カラム値

テキストデータの抽出

CLOB カラムに XML や JSON が格納されている場合:

  1. LOB カラム: XML_CONTENT (CLOB) を選択
  2. 拡張子: xml
注意: 大量の LOB レコードがある場合、抽出には時間がかかります。 進捗ダイアログでキャンセルできます。
注意: テーブルプレビューでは LOB データは %BLOB%、 %CLOB%、%NCLOB% とプレースホルダー表示されます。 実データを取得するにはこの LOB ファイル抽出機能を使用してください。
