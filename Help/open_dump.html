ダンプファイルを開く

操作手順

  1. メニューの ファイル(&F) > ダンプファイルを開く(&O)... を選択します。
    または、ツールバーの 開く ボタンをクリックします。
  2. ファイル選択ダイアログで、ダンプファイル (.dmp) を選択します。
  3. 解析が開始され、進捗バーが表示されます。
  4. 完了すると、テーブル一覧がワークスペースに表示されます。

自動検出

ダンプファイルの形式 (EXP / EXPDP / EXPDP 圧縮) は自動的に検出されます。 ファイルの先頭バイトおよびマーカーバイトを解析して判定します。

形式検出方法
EXP先頭バイト 0x01-0x05 + "EXPORT:" 文字列
EXPDP"xml version" 文字列
EXPDP (圧縮)"KGC" + "HDR" マーカー

文字セット

ダンプファイルの文字セットも自動検出されます。 対応する文字セット:

  • AL32UTF8 / UTF8
  • JA16SJIS (Shift_JIS)
  • JA16EUC (EUC-JP)
  • AL16UTF16
  • US7ASCII
  • WE8ISO8859P1 / WE8MSWIN1252
注意: 圧縮された EXPDP ダンプの場合、テーブル定義 (DDL) のみ読み取り可能です。 データレコードの解凍には対応していません。
