Verificando a Versão

Selecione Ajuda(&H) > Sobre(&A) no menu para verificar a versão atual.

A caixa de diálogo Sobre compara automaticamente com a versão mais recente do GitHub e indica se uma atualização está disponível.

  • Se estiver usando a versão mais recente: o número da versão é exibido em verde
  • Se uma versão mais nova estiver disponível: o número da versão é exibido em laranja

Atualização Automática

Quando uma nova versão está disponível, um botão de atualização aparece na caixa de diálogo.

  1. Clique no botão Atualizar para a Versão Mais Recente na caixa de diálogo Sobre.
  2. Selecione Sim na caixa de diálogo de confirmação.
  3. O instalador MSI mais recente é baixado automaticamente (uma barra de progresso é exibida).
  4. Após o download, o aplicativo é fechado e o instalador é iniciado automaticamente.

Você também pode clicar no link Abrir Página de Lançamento para visualizar a página de lançamento do GitHub no seu navegador.

Relatório de Bugs

Se você encontrar um bug ou problema, reporte-o em Ajuda(&H) > Relatório de Bug(&R).

As seguintes informações são coletadas automaticamente e exibidas de forma transparente na caixa de diálogo de relatório:

  • Versão do aplicativo / Versão da DLL
  • Versão do SO / Versão do .NET
  • Arquitetura / Resolução de tela / DPI
  • Informações do arquivo de dump (somente tamanho e formato; caminhos não são enviados)

Os relatórios são enviados automaticamente como GitHub Issues por meio de um Cloudflare Worker. Nenhuma informação pessoal (nome de usuário, nome do PC, caminhos de arquivos, etc.) é coletada ou enviada.

Se o arquivo de dump tiver 50 MB ou menos, você pode opcionalmente anexá-lo ao relatório.

Suporte Multilíngue

10 idiomas são suportados: japonês, inglês, chinês (simplificado), coreano, alemão, francês, espanhol, italiano, russo e português (BR). O idioma é selecionado automaticamente com base no idioma de exibição do sistema na inicialização.
