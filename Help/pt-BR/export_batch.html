Exportação em Lote

Visão Geral

Exporte múltiplas tabelas para arquivos CSV ou SQL de uma só vez. Cada tabela é gerada como um arquivo individual.

Passos

  1. Clique no botão Exportação em Lote na barra de ferramentas.
  2. A caixa de diálogo de exportação é exibida.
  3. Selecione o formato de saída (CSV / SQL).
  4. Especifique a pasta de saída.
  5. Confirme as tabelas a serem exportadas.
    Desmarque as tabelas que não são necessárias.
  6. Clique no botão Exportar.

Nomes dos Arquivos de Saída

Os nomes dos arquivos são gerados automaticamente no formato NomeDoSchema.NomeDaTabela.csv ou NomeDoSchema.NomeDaTabela.sql.

Seleção / Exclusão de Tabelas

  • Botões Selecionar Tudo / Desmarcar Tudo para operações em massa
  • Inverter Seleção para alternar os estados de seleção
  • Clique em caixas de seleção individuais para selecionar tabelas específicas
Dica: As tabelas excluídas na área de trabalho também são automaticamente excluídas da exportação em lote.

Saída de Relatórios

Em Objeto(&O) > Saída de Relatório(&R), você também pode gerar informações de tabelas em arquivos de texto.

  • Lista de Objetos — Gera uma lista em texto com nomes de tabelas e contagem de linhas
  • Definições de Tabelas — Gera as definições de colunas (nome, tipo, tamanho) de todas as tabelas
