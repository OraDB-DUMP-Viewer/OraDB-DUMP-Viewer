Exportação CSV

Passos

  1. Selecione a tabela que deseja exportar na lista de tabelas.
  2. Clique no botão CSV na barra de ferramentas, ou selecione Objeto(&O) > Exportar > Dados(&D).
  3. Especifique o caminho do arquivo de destino.
  4. A exportação é executada e uma caixa de diálogo de progresso é exibida.

Formato de Saída

  • Codificação: UTF-8 (com BOM)
  • Delimitador: Vírgula (,)
  • Aspas: Aspas duplas (") — aplicadas quando os valores contêm vírgulas, quebras de linha ou aspas duplas
  • Fim de linha: CRLF (\r\n)
  • Compatível com RFC 4180

Opções

As seguintes opções podem ser configuradas em Opções de Exportação:

  • Linha de Cabeçalho — Se os nomes das colunas devem ser exibidos na primeira linha
  • Linha de Informação de Tipo — Exibir informações de tipo das colunas na linha após o cabeçalho
  • Formato de Data — Formato de saída para colunas DATE/TIMESTAMP

Outros Formatos de Exportação

Exportação Excel

Exporta dados da tabela para um arquivo Excel (.xlsx / .xls).

  1. Selecione uma tabela e clique no botão Excel na barra de ferramentas.
  2. Especifique o caminho do arquivo de destino.
  3. Um arquivo Excel é gerado.

Exportação Access

Exporta dados da tabela para um banco de dados Microsoft Access (.accdb / .mdb).

  1. Selecione uma tabela e clique no botão Access na barra de ferramentas.
  2. Especifique o caminho do arquivo de destino.
  3. Uma tabela é criada no banco de dados Access e os dados são importados.

Exportação SQL Server

Exporta dados da tabela diretamente para um banco de dados SQL Server.

  1. Selecione uma tabela e clique no botão SQL Server na barra de ferramentas.
  2. Insira as informações de conexão do SQL Server na caixa de diálogo de conexão.
  3. Uma tabela é criada no banco de dados de destino e os dados são inseridos.

Se uma tabela com o mesmo nome já existir, ela será automaticamente excluída e recriada.

Exportação ODBC

Exporta dados da tabela para qualquer banco de dados compatível com ODBC.

  1. Selecione uma tabela e clique no botão ODBC na barra de ferramentas.
  2. Insira a string de conexão ODBC ou DSN na caixa de diálogo de conexão.
  3. Uma tabela é criada no banco de dados de destino e os dados são inseridos.

Se uma tabela com o mesmo nome já existir, ela será automaticamente excluída e recriada.

Tratamento de Tabelas Existentes

Destino da ExportaçãoComportamento com Tabela Existente
Script SQLDROP TABLE IF EXISTS (alternável via caixa de seleção)
SQL ServerExclusão automática + recriação
ODBCExclusão automática + recriação
AccessRecriar o arquivo inteiro
ExcelSobrescrever o arquivo
Nota: Para exportações SQL Server e ODBC, o banco de dados de destino deve ser criado antecipadamente.
