Exportação SQL

Passos

  1. Selecione a tabela que deseja exportar na lista de tabelas.
  2. Clique no botão SQL na barra de ferramentas, ou selecione Objeto(&O) > Exportar > Script(&S).
  3. Especifique o caminho do arquivo de destino.
  4. A exportação é executada.

SGBD Suportados

Instruções INSERT podem ser geradas para os seguintes SGBDs:

SGBDDiferenças de Sintaxe
OracleFunção TO_DATE, tipo NUMBER mantido
PostgreSQLSintaxe CAST, tipo TIMESTAMP
MySQLIdentificadores com crase, tipo DATETIME
SQL ServerIdentificadores com colchetes, tipo DATETIME2

Opções

Uma caixa de diálogo é exibida antes da exportação ao clicar no botão SQL. As seguintes opções podem ser configuradas:

  • Tipo de SGBD — Escolha entre Oracle / PostgreSQL / MySQL / SQL Server
  • Saída DROP TABLE + CREATE TABLE (padrão: ATIVADO) — Alternável via caixa de seleção. Quando ativado, gera DROP TABLE IF EXISTS e CREATE TABLE DDL antes das instruções INSERT. Exclui e recria a tabela se ela já existir no banco de dados de destino.
  • Formato de Data — Formato literal ou chamada de função

Sintaxe DROP TABLE por SGBD

SGBDSintaxe
Oracle / PostgreSQLDROP TABLE IF EXISTS "table_name" CASCADE;
MySQLDROP TABLE IF EXISTS `table_name`;
SQL ServerIF OBJECT_ID('table_name', 'U') IS NOT NULL DROP TABLE [table_name];
Nota: Dados LOB (BLOB/CLOB) não são incluídos nas exportações SQL. Use a Extração de Arquivos LOB para exportar dados LOB.
