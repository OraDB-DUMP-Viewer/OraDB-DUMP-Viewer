O que é o OraDB DUMP Viewer?

O OraDB DUMP Viewer é uma ferramenta autônoma que permite navegar e exportar arquivos de dump de banco de dados Oracle (EXP / EXPDP) sem um cliente Oracle.

Principais Recursos

  • Análise de Arquivos de Dump — Suporta os formatos EXP (legado) e EXPDP (Data Pump)
  • Lista de Tabelas — Exibe schemas, nomes de tabelas e informações de colunas em uma visualização em árvore
  • Visualização de Dados — Exibição paginada dos dados da tabela com suporte a ordenação e cópia
  • Pesquisa Avançada — Filtre dados usando 12 operadores com condições compostas AND/OR
  • Exportação CSV — Exporte dados de tabelas para arquivos CSV
  • Exportação SQL — Gere instruções INSERT (Oracle / PostgreSQL / MySQL / SQL Server)
  • Exportação Excel — Exporte para o formato .xlsx / .xls
  • Exportação Access — Exporte para bancos de dados .mdb / .accdb
  • Exportação SQL Server — Exporte diretamente para um banco de dados SQL Server
  • Exportação ODBC — Exporte para qualquer banco de dados compatível com ODBC
  • Exportação em Lote — Exporte múltiplas tabelas para CSV/SQL de uma só vez
  • Extração de Arquivos LOB — Extraia dados BLOB/CLOB/NCLOB para arquivos individuais
  • Área de Trabalho — Salve e restaure o estado do seu trabalho
  • Exclusão de Tabelas — Oculte tabelas indesejadas (com suporte a desfazer/refazer)
  • Suporte Multilíngue — 10 idiomas incluindo japonês, inglês, chinês, coreano, alemão, francês, espanhol, italiano, russo e português (BR)
  • Relatório de Bugs — Reporte bugs diretamente para o GitHub Issues de dentro do aplicativo
  • Atualização Automática — Detecta novas versões e oferece atualização com um clique

Formatos de Dump Suportados

FormatoFerramentaStatus de Suporte
EXP (Legado)exp / impSuportado (Oracle 7–11g)
EXPDP (Data Pump)expdp / impdpSuportado (Oracle 10g–23ai)
EXPDP Comprimidoexpdp COMPRESSION=ALLParcial (somente definições de tabelas)

Instalação

  1. Execute o instalador MSI distribuído.
  2. Siga as instruções do assistente de instalação.
  3. Inicie o "OraDB DUMP Viewer" a partir da área de trabalho ou do menu Iniciar.
