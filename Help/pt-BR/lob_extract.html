Extração de Arquivos LOB

Visão Geral

Extrai dados armazenados em colunas BLOB / CLOB / NCLOB de uma tabela como um arquivo por registro.

Tipos de Dados Suportados

Tipo de ColunaConteúdo de Saída
BLOBArquivos binários (imagens, PDF, ZIP, etc.)
CLOBArquivos de texto (decodificados usando o conjunto de caracteres do arquivo de dump)
NCLOBArquivos de texto Unicode

Passos

  1. Selecione uma tabela que contenha colunas LOB na lista de tabelas.
  2. Selecione Arquivo(&F) > Extrair Arquivos(&F) no menu.
  3. A caixa de diálogo de extração LOB é exibida.
  4. Configure o seguinte:
    • Coluna LOB — Selecione a coluna LOB a ser extraída
    • Pasta de Saída — Diretório para os arquivos de saída
    • Nomenclatura de Arquivo — Numeração sequencial ou valor de coluna
    • Extensão — Extensão do arquivo de saída (padrão: lob)
  5. Clique no botão Extrair.

Métodos de Nomenclatura de Arquivo

MétodoExemploDescrição
Sequencial000001.jpg, 000002.jpgNúmeros sequenciais de 6 dígitos começando em 1
Valor de Colunavalor_coluna_nome.jpgUsa o valor de uma coluna especificada como nome do arquivo

Exemplos de Uso

Extraindo Arquivos de Imagem

Quando imagens JPEG estão armazenadas em uma coluna BLOB:

  1. Coluna LOB: Selecione IMAGE_DATA (BLOB)
  2. Extensão: jpg
  3. Nomenclatura de Arquivo: Sequencial ou valor da coluna ID

Extraindo Dados de Texto

Quando XML ou JSON está armazenado em uma coluna CLOB:

  1. Coluna LOB: Selecione XML_CONTENT (CLOB)
  2. Extensão: xml
Atenção: A extração pode demorar quando há muitos registros LOB. Você pode cancelar pela caixa de diálogo de progresso.
Nota: Na visualização da tabela, os dados LOB são exibidos como marcadores: %BLOB%, %CLOB%, %NCLOB%. Use este recurso de Extração de Arquivos LOB para obter os dados reais.
