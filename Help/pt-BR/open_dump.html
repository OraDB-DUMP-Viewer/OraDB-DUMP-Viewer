Abrindo um Arquivo de Dump

Passos

  1. Selecione Arquivo(&F) > Abrir Arquivo de Dump(&O)... no menu.
    Ou clique no botão Abrir na barra de ferramentas.
  2. Na caixa de diálogo de arquivo, selecione um arquivo de dump (.dmp).
  3. A análise começa e uma barra de progresso é exibida.
  4. Ao concluir, a lista de tabelas aparece na área de trabalho.

Detecção Automática

O formato do arquivo de dump (EXP / EXPDP / EXPDP Comprimido) é detectado automaticamente analisando os bytes iniciais e os bytes marcadores do arquivo.

FormatoMétodo de Detecção
EXPBytes iniciais 0x01-0x05 + string "EXPORT:"
EXPDPString "xml version"
EXPDP (Comprimido)Marcadores "KGC" + "HDR"

Conjunto de Caracteres

O conjunto de caracteres do arquivo de dump também é detectado automaticamente. Conjuntos de caracteres suportados:

  • AL32UTF8 / UTF8
  • JA16SJIS (Shift_JIS)
  • JA16EUC (EUC-JP)
  • AL16UTF16
  • US7ASCII
  • WE8ISO8859P1 / WE8MSWIN1252
Nota: Para dumps EXPDP comprimidos, somente as definições de tabelas (DDL) podem ser lidas. A descompressão de registros de dados não é suportada.
