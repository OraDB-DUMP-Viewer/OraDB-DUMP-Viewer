Limitações

  • Registros de dados em dumps EXPDP comprimidos não podem ser lidos (somente metadados)
  • Os dados reais do tipo BFILE não podem ser extraídos (somente informações de caminho)
  • Arquivos de dump criptografados não são suportados
  • Dumps via link de rede não são suportados
  • Número máximo de tabelas: 1.000
  • Número máximo de colunas: 1.000 (por tabela)

Conjuntos de Caracteres

Os seguintes conjuntos de caracteres NLS são suportados:

  • AL32UTF8 / UTF8
  • JA16SJIS
  • JA16EUC
  • AL16UTF16
  • US7ASCII
  • WE8ISO8859P1 / WE8MSWIN1252
