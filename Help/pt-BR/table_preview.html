Visualização de Tabela

Abrindo uma Tabela

Clique duas vezes em um nome de tabela na lista de tabelas para abrir a visualização de dados. Ou clique com o botão direito na tabela e selecione Abrir(&O).

Exibição de Dados

Os dados são exibidos em um DataGridView.

  • Os nomes das colunas aparecem nos cabeçalhos das colunas.
  • Cada linha exibe um registro.
  • Valores NULL são exibidos em branco.
  • Dados LOB (BLOB/CLOB/NCLOB) são exibidos como %BLOB%, %CLOB%, %NCLOB%.

Paginação

Quando há uma grande quantidade de dados, eles são exibidos página por página.

  • Linhas por página — Ajustável pelo campo numérico no canto superior direito (padrão: 100 linhas)
  • Informação da página — O número da página atual e o total de páginas são exibidos
  • Use os botões de navegação para alternar entre as páginas

Ordenação

Clique no cabeçalho de uma coluna para ordenar os dados por essa coluna. Clique novamente para alternar entre ordem crescente e decrescente.

Copiando Células

Selecione as células na visualização e pressione Ctrl+C para copiar para a área de transferência. A seleção e cópia de múltiplas células também é suportada.

Pesquisa Avançada

Clique no botão Pesquisa Avançada na tela de visualização para filtrar dados usando 12 operadores com condições compostas AND/OR. Consulte Pesquisa & Filtro para detalhes.

Exportando a partir da Visualização

As seguintes exportações estão disponíveis na barra de ferramentas da visualização:

  • SQL — Saída de script SQL
  • CSV — Saída de arquivo CSV
  • Excel — Saída de arquivo Excel
  • Access — Saída para banco de dados Access
  • SQL Server — Saída direta para SQL Server
  • ODBC — Saída para um banco de dados compatível com ODBC

Propriedades da Tabela

Clique com o botão direito em uma tabela e selecione Propriedades(&P) para visualizar as definições de colunas (nome da coluna, tipo de dados, tamanho, etc.).

Exclusão de Tabelas

Você pode ocultar temporariamente tabelas indesejadas.

  1. Selecione tabelas na lista de tabelas (seleção múltipla é suportada).
  2. Escolha Excluir(&D) no menu de contexto, ou pressione a tecla Delete.
  3. As tabelas excluídas são exibidas na barra de status.

Para desfazer a exclusão:

  • Editar(&E) > Desfazer (Ctrl+Z)
  • Objeto(&O) > Limpar Todos os Filtros(&F) para remover todas as exclusões
