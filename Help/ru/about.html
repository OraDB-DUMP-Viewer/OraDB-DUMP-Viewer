Проверка версии

Выберите Справка(&H) > О программе(&A) в меню, чтобы проверить текущую версию.

Диалог «О программе» автоматически сравнивает версию с последним релизом на GitHub и показывает, доступно ли обновление.

  • Если используется последняя версия: номер версии отображается зелёным цветом
  • Если доступна более новая версия: номер версии отображается оранжевым цветом

Автоматическое обновление

Когда доступна новая версия, в диалоге появляется кнопка обновления.

  1. Нажмите кнопку Обновить до последней версии в диалоге «О программе».
  2. Выберите Да в диалоге подтверждения.
  3. Последний MSI-установщик загружается автоматически (отображается индикатор прогресса).
  4. После загрузки приложение закрывается, и установщик запускается автоматически.

Вы также можете нажать ссылку Открыть страницу релиза, чтобы просмотреть страницу релиза на GitHub в браузере.

Активация лицензии

При первом запуске требуется активация лицензии. Получите лицензию на https://www.odv.dev/.

ПланСтоимостьНазначение
Личное использованиеБесплатноЧастные лица
ОбразовательныйБесплатноУчебные заведения
Корпоративное использование¥9 800/год (без учёта налога)Компании и коммерческое использование

Проверка лицензии включает локальную верификацию подписи RSA и онлайн-проверку при запуске. В автономных средах доступна только локальная проверка.

Повторную аутентификацию или изменение лицензионного ключа можно выполнить через Справка(&H) > Активация лицензии.

Сообщения об ошибках

Если вы обнаружили ошибку или проблему, сообщите о ней через Справка(&H) > Сообщить об ошибке(&R).

Следующая информация автоматически собирается и отображается в диалоге отчёта:

  • Версия приложения / версия DLL
  • Версия ОС / версия .NET
  • Архитектура / разрешение экрана / DPI
  • Информация о файле дампа (только размер и формат; пути не отправляются)

Отчёты автоматически отправляются как GitHub Issues через Cloudflare Worker. Личная информация (имя пользователя, имя ПК, пути к файлам и т. д.) не собирается и не отправляется.

Если размер файла дампа не превышает 50 МБ, его можно при желании прикрепить к отчёту.

Многоязычная поддержка

Поддерживается 10 языков: японский, английский, китайский (упрощённый), корейский, немецкий, французский, испанский, итальянский, русский и португальский (BR). Язык автоматически выбирается на основе языка отображения системы при запуске.
