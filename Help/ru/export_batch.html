Пакетный экспорт

Обзор

Экспорт нескольких таблиц в файлы CSV или SQL за один раз. Каждая таблица сохраняется в отдельный файл.

Порядок действий

  1. Нажмите кнопку Пакетный экспорт на панели инструментов.
  2. Отобразится диалог экспорта.
  3. Выберите формат вывода (CSV / SQL).
  4. Укажите папку для вывода.
  5. Подтвердите список таблиц для экспорта.
    Снимите флажки с ненужных таблиц.
  6. Нажмите кнопку Экспорт.

Имена выходных файлов

Имена файлов генерируются автоматически в формате ИмяСхемы.ИмяТаблицы.csv или ИмяСхемы.ИмяТаблицы.sql.

Выбор / исключение таблиц

  • Кнопки Выбрать все / Снять все для массовых операций
  • Инвертировать выделение для переключения состояния выбора
  • Нажимайте отдельные флажки для выбора конкретных таблиц
Совет: Таблицы, исключённые в рабочей области, автоматически исключаются и из пакетного экспорта.

Вывод отчётов

Через Объект(&O) > Вывод отчёта(&R) можно также вывести информацию о таблицах в текстовые файлы.

  • Список объектов — Вывод текстового списка имён таблиц и количества строк
  • Определения таблиц — Вывод определений столбцов (имя, тип, размер) для всех таблиц
