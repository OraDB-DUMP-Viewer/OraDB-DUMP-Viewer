Экспорт в CSV

Порядок действий

  1. Выберите таблицу для экспорта в списке таблиц.
  2. Нажмите кнопку CSV на панели инструментов или выберите Объект(&O) > Экспорт > Данные(&D).
  3. Укажите путь к файлу назначения.
  4. Запустится экспорт, и отобразится диалог прогресса.

Формат вывода

  • Кодировка: UTF-8 (с BOM)
  • Разделитель: Запятая (,)
  • Кавычки: Двойные кавычки (") — применяются, если значения содержат запятые, переводы строк или двойные кавычки
  • Окончание строки: CRLF (\r\n)
  • Соответствует RFC 4180

Параметры

Следующие параметры можно настроить в разделе Параметры экспорта:

  • Строка заголовка — Выводить ли имена столбцов в первой строке
  • Строка типов — Выводить информацию о типах столбцов в строке после заголовка
  • Формат даты — Формат вывода для столбцов DATE/TIMESTAMP

Другие форматы экспорта

Экспорт в Excel

Экспортирует данные таблицы в файл Excel (.xlsx / .xls).

  1. Выберите таблицу и нажмите кнопку Excel на панели инструментов.
  2. Укажите путь к файлу назначения.
  3. Будет создан файл Excel.

Экспорт в Access

Экспортирует данные таблицы в базу данных Microsoft Access (.accdb / .mdb).

  1. Выберите таблицу и нажмите кнопку Access на панели инструментов.
  2. Укажите путь к файлу назначения.
  3. В базе данных Access создаётся таблица, и данные импортируются.

Экспорт в SQL Server

Экспортирует данные таблицы непосредственно в базу данных SQL Server.

  1. Выберите таблицу и нажмите кнопку SQL Server на панели инструментов.
  2. Введите информацию о подключении к SQL Server в диалоге подключения.
  3. В целевой базе данных создаётся таблица, и данные вставляются.

Если таблица с таким именем уже существует, она автоматически удаляется и создаётся заново.

Экспорт через ODBC

Экспортирует данные таблицы в любую базу данных, совместимую с ODBC.

  1. Выберите таблицу и нажмите кнопку ODBC на панели инструментов.
  2. Введите строку подключения ODBC или DSN в диалоге подключения.
  3. В целевой базе данных создаётся таблица, и данные вставляются.

Если таблица с таким именем уже существует, она автоматически удаляется и создаётся заново.

Обработка существующих таблиц

Цель экспортаПоведение при наличии таблицы
SQL-скриптDROP TABLE IF EXISTS (переключается флажком)
SQL ServerАвтоматическое удаление и пересоздание
ODBCАвтоматическое удаление и пересоздание
AccessПересоздание всего файла
ExcelПерезапись файла
Примечание: Для экспорта в SQL Server и ODBC целевая база данных должна быть создана заранее.
