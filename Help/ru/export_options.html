Параметры экспорта

Как открыть

Нажмите кнопку Параметры на панели инструментов, чтобы открыть диалог параметров экспорта.

ФорматПримерОписание
YYYY/MM/DD HH:MI:SS2026/01/15 09:30:00Стандартный формат (по умолчанию)
YYYYMMDD20260115Компактный (только дата)
YYYYMMDDHH24MISS20260115093000Компактный (дата и время)
ПользовательскийЛюбойТокены: YYYY, MM, DD, HH24, MI, SS

Параметры SQL

Перед экспортом SQL отображается диалоговое окно. Следующие параметры переключаются флажками:

НастройкаПо умолчаниюОписание
Выводить DROP TABLE + CREATE TABLEВКЛВыводить DROP TABLE IF EXISTS и CREATE TABLE DDL перед операторами INSERT. Переключается флажком. Удаляет и пересоздаёт существующие таблицы
Тип СУБДOracleOracle / PostgreSQL / MySQL / SQL Server
Примечание: Настройки параметров сохраняются в настройках приложения и сохраняются между перезапусками.
