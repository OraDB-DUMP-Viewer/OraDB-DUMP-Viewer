Экспорт в SQL

Порядок действий

  1. Выберите таблицу для экспорта в списке таблиц.
  2. Нажмите кнопку SQL на панели инструментов или выберите Объект(&O) > Экспорт > Скрипт(&S).
  3. Укажите путь к файлу назначения.
  4. Запустится экспорт.

Поддерживаемые СУБД

Операторы INSERT могут быть сгенерированы для следующих СУБД:

СУБДОсобенности синтаксиса
OracleФункция TO_DATE, тип NUMBER без изменений
PostgreSQLСинтаксис CAST, тип TIMESTAMP
MySQLИдентификаторы в обратных кавычках, тип DATETIME
SQL ServerИдентификаторы в квадратных скобках, тип DATETIME2

Параметры

Перед экспортом при нажатии кнопки SQL отображается диалоговое окно. Можно настроить следующие параметры:

  • Тип СУБД — Выбор из Oracle / PostgreSQL / MySQL / SQL Server
  • Вывод DROP TABLE + CREATE TABLE (по умолчанию: ВКЛ) — Переключается флажком. При включении выводятся DROP TABLE IF EXISTS и CREATE TABLE DDL перед операторами INSERT. Удаляет и пересоздаёт таблицу, если она уже существует в целевой базе данных.
  • Формат даты — Литеральный формат или вызов функции

Синтаксис DROP TABLE по СУБД

СУБДСинтаксис
Oracle / PostgreSQLDROP TABLE IF EXISTS "table_name" CASCADE;
MySQLDROP TABLE IF EXISTS `table_name`;
SQL ServerIF OBJECT_ID('table_name', 'U') IS NOT NULL DROP TABLE [table_name];
Примечание: Данные LOB (BLOB/CLOB) не включаются в SQL-экспорт. Используйте Извлечение файлов LOB для экспорта данных LOB.
