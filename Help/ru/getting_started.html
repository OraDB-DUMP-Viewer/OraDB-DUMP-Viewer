Что такое OraDB DUMP Viewer?

OraDB DUMP Viewer — автономный инструмент для просмотра и экспорта файлов дампов баз данных Oracle (EXP / EXPDP) без необходимости установки клиента Oracle.

Основные возможности

  • Анализ файлов дампов — Поддержка форматов EXP (устаревший) и EXPDP (Data Pump)
  • Список таблиц — Отображение схем, имён таблиц и информации о столбцах в виде дерева
  • Предварительный просмотр данных — Постраничное отображение данных таблиц с сортировкой и возможностью копирования
  • Расширенный поиск — Фильтрация данных с использованием 12 операторов и составных условий AND/OR
  • Экспорт в CSV — Экспорт данных таблиц в файлы CSV
  • Экспорт в SQL — Генерация операторов INSERT (Oracle / PostgreSQL / MySQL / SQL Server)
  • Экспорт в Excel — Экспорт в формат .xlsx / .xls
  • Экспорт в Access — Экспорт в базы данных .mdb / .accdb
  • Экспорт в SQL Server — Прямой экспорт в базу данных SQL Server
  • Экспорт через ODBC — Экспорт в любую базу данных, совместимую с ODBC
  • Пакетный экспорт — Одновременный экспорт нескольких таблиц в CSV/SQL
  • Извлечение файлов LOB — Извлечение данных BLOB/CLOB/NCLOB в отдельные файлы
  • Рабочая область — Сохранение и восстановление рабочего состояния
  • Исключение таблиц — Скрытие ненужных таблиц (с поддержкой отмены/повтора)
  • Многоязычная поддержка — 10 языков: японский, английский, китайский, корейский, немецкий, французский, испанский, итальянский, русский и португальский (BR)
  • Сообщения об ошибках — Отправка отчётов об ошибках непосредственно в GitHub Issues из приложения
  • Автоматическое обновление — Обнаружение новых версий и обновление в один клик

Поддерживаемые форматы дампов

ФорматИнструментСтатус поддержки
EXP (устаревший)exp / impПоддерживается (Oracle 7–11g)
EXPDP (Data Pump)expdp / impdpПоддерживается (Oracle 10g–23ai)
EXPDP со сжатиемexpdp COMPRESSION=ALLЧастично (только определения таблиц)

Установка

  1. Запустите предоставленный MSI-установщик.
  2. Следуйте инструкциям мастера установки.
  3. Запустите «OraDB DUMP Viewer» с рабочего стола или из меню «Пуск».
