Извлечение файлов LOB

Обзор

Извлекает данные, хранящиеся в столбцах BLOB / CLOB / NCLOB таблицы, в виде одного файла на каждую запись.

Поддерживаемые типы данных

Тип столбцаСодержимое вывода
BLOBДвоичные файлы (изображения, PDF, ZIP и т. д.)
CLOBТекстовые файлы (декодируются с использованием кодировки файла дампа)
NCLOBТекстовые файлы в кодировке Unicode

Порядок действий

  1. Выберите таблицу, содержащую столбцы LOB, в списке таблиц.
  2. Выберите Файл(&F) > Извлечь файлы(&F) в меню.
  3. Отобразится диалог извлечения LOB.
  4. Настройте следующие параметры:
    • Столбец LOB — Выберите столбец LOB для извлечения
    • Папка вывода — Каталог для выходных файлов
    • Именование файлов — Последовательная нумерация или значение столбца
    • Расширение — Расширение выходных файлов (по умолчанию: lob)
  5. Нажмите кнопку Извлечь.

Методы именования файлов

МетодПримерОписание
Последовательный000001.jpg, 000002.jpg6-значные последовательные номера, начиная с 1
Значение столбцаfilename_col_value.jpgИспользует значение указанного столбца в качестве имени файла

Примеры использования

Извлечение файлов изображений

Когда изображения JPEG хранятся в столбце BLOB:

  1. Столбец LOB: Выберите IMAGE_DATA (BLOB)
  2. Расширение: jpg
  3. Именование файлов: Последовательное или значение столбца ID

Извлечение текстовых данных

Когда XML или JSON хранятся в столбце CLOB:

  1. Столбец LOB: Выберите XML_CONTENT (CLOB)
  2. Расширение: xml
Внимание: Извлечение может занять значительное время при большом количестве записей LOB. Вы можете отменить операцию через диалог прогресса.
Примечание: В предварительном просмотре таблицы данные LOB отображаются как заполнители: %BLOB%, %CLOB%, %NCLOB%. Используйте функцию извлечения файлов LOB для получения фактических данных.
