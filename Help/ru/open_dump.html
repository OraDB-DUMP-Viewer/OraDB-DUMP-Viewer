Открытие файла дампа

Порядок действий

  1. Выберите Файл(&F) > Открыть файл дампа(&O)... в меню.
    Или нажмите кнопку Открыть на панели инструментов.
  2. В диалоговом окне выберите файл дампа (.dmp).
  3. Начнётся анализ, и отобразится индикатор прогресса.
  4. По завершении список таблиц появится в рабочей области.

Автоматическое определение формата

Формат файла дампа (EXP / EXPDP / EXPDP со сжатием) определяется автоматически путём анализа начальных байтов и маркерных байтов файла.

ФорматМетод определения
EXPНачальные байты 0x01-0x05 + строка "EXPORT:"
EXPDPСтрока "xml version"
EXPDP (со сжатием)Маркеры "KGC" + "HDR"

Кодировка символов

Кодировка символов файла дампа также определяется автоматически. Поддерживаемые кодировки:

  • AL32UTF8 / UTF8
  • JA16SJIS (Shift_JIS)
  • JA16EUC (EUC-JP)
  • AL16UTF16
  • US7ASCII
  • WE8ISO8859P1 / WE8MSWIN1252
Примечание: Для сжатых дампов EXPDP доступно чтение только определений таблиц (DDL). Распаковка записей данных не поддерживается.
