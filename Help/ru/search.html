Поиск и фильтрация

Фильтр по имени таблицы

Введите часть имени таблицы в текстовое поле поиска в верхней части рабочей области, чтобы отфильтровать список таблиц в реальном времени.

  • Без учёта регистра.
  • Частичное совпадение по имени таблицы.
  • Очистите текст, чтобы снять фильтр.

Расширенный поиск (поиск по данным таблицы)

На экране предварительного просмотра таблицы можно выполнять расширенный поиск данных по условиям. Нажмите кнопку Расширенный поиск, чтобы открыть диалог настройки условий поиска.

Настройка условий поиска

Для каждой строки условия укажите следующее:

  1. Имя столбца — Выберите целевой столбец
  2. Оператор — Выберите один из 12 операторов
  3. Значение поиска — Введите значение для сравнения (не требуется для операторов Null / Not Null)
  4. Учёт регистра — Установите флажок для сравнения с учётом регистра

Составные условия (AND / OR)

При добавлении условий можно указать логический оператор между ними: AND или OR.

  • AND — Показывать только строки, удовлетворяющие всем условиям
  • OR — Показывать строки, удовлетворяющие любому из условий

Строки условий можно свободно добавлять и удалять, что позволяет создавать сложные фильтры.

Сохранение условий поиска

Условия поиска автоматически восстанавливаются при повторном открытии той же таблицы. Чтобы сбросить условия, используйте кнопку Очистить поиск на экране предварительного просмотра.

Инвертировать выделение

Можно инвертировать состояние выделения в списке таблиц. Используйте пункт Инвертировать выделение в контекстном меню.
