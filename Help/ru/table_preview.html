Предварительный просмотр таблиц

Открытие таблицы

Дважды щёлкните имя таблицы в списке таблиц, чтобы открыть предварительный просмотр данных. Или щёлкните правой кнопкой мыши по таблице и выберите Открыть(&O).

Отображение данных

Данные отображаются в DataGridView.

  • Имена столбцов отображаются в заголовках.
  • Каждая строка соответствует одной записи.
  • Значения NULL отображаются как пустые.
  • Данные LOB (BLOB/CLOB/NCLOB) отображаются как %BLOB%, %CLOB%, %NCLOB%.

Постраничный просмотр

При большом объёме данных они отображаются постранично.

  • Строк на странице — Настраивается через числовое поле в правом верхнем углу (по умолчанию: 100 строк)
  • Информация о странице — Отображаются номер текущей страницы и общее количество страниц
  • Используйте кнопки навигации для переключения между страницами

Сортировка

Нажмите на заголовок столбца для сортировки данных по этому столбцу. Повторное нажатие переключает между сортировкой по возрастанию и убыванию.

Копирование ячеек

Выделите ячейки в области просмотра и нажмите Ctrl+C, чтобы скопировать в буфер обмена. Также поддерживается выделение и копирование нескольких ячеек.

Расширенный поиск

Нажмите кнопку Расширенный поиск на экране предварительного просмотра для фильтрации данных с использованием 12 операторов и составных условий AND/OR. Подробнее см. Поиск и фильтрация.

Экспорт из предварительного просмотра

На панели инструментов предварительного просмотра доступны следующие варианты экспорта:

  • SQL — Вывод SQL-скрипта
  • CSV — Вывод в файл CSV
  • Excel — Вывод в файл Excel
  • Access — Вывод в базу данных Access
  • SQL Server — Прямой вывод в SQL Server
  • ODBC — Вывод в базу данных, совместимую с ODBC

Свойства таблицы

Щёлкните правой кнопкой мыши по таблице и выберите Свойства(&P) для просмотра определений столбцов (имя столбца, тип данных, длина и т. д.).

Исключение таблиц

Вы можете временно скрыть ненужные таблицы.

  1. Выберите таблицы в списке (поддерживается множественный выбор).
  2. Выберите Удалить(&D) из контекстного меню или нажмите клавишу Delete.
  3. Исключённые таблицы отображаются в строке состояния.

Чтобы отменить исключение:

  • Правка(&E) > Отменить (Ctrl+Z)
  • Объект(&O) > Сбросить все фильтры(&F) для удаления всех исключений
