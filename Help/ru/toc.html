Справка OraDB DUMP Viewer

OraDB DUMP Viewer — автономный инструмент для просмотра и экспорта файлов дампов Oracle Data Pump (EXPDP) и устаревшего формата EXP через графический интерфейс без необходимости установки клиента Oracle.
