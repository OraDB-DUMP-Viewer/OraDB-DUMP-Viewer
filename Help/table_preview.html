テーブルプレビュー

テーブルを開く

テーブル一覧でテーブル名をダブルクリックすると、データプレビュー画面が開きます。 または、テーブルを右クリックして 開く(&O) を選択します。

データ表示

データは DataGridView で一覧表示されます。

  • 列ヘッダーにカラム名が表示されます。
  • 各行に1レコードのデータが表示されます。
  • NULL 値は空白で表示されます。
  • LOB データ (BLOB/CLOB/NCLOB) は %BLOB%, %CLOB%, %NCLOB% と表示されます。

ページング

データが大量にある場合、ページ単位で表示されます。

  • 1ページあたりの行数 — 画面右上の数値入力で変更可能 (デフォルト: 100行)
  • ページ情報 — 現在のページ番号と総ページ数が表示されます
  • ナビゲーションボタンでページを切り替えます

ソート

列ヘッダーをクリックすると、その列でデータをソートできます。 再度クリックすると昇順/降順が切り替わります。

セルのコピー

プレビュー上のセルを選択し、Ctrl+C でクリップボードにコピーできます。 複数セルの選択・コピーにも対応しています。

高度な検索

プレビュー画面の 高度な検索 ボタンをクリックすると、 12種類の演算子と AND/OR 複合条件でデータをフィルタリングできます。 詳しくは 検索・フィルタ を参照してください。

プレビューからのエクスポート

プレビュー画面のツールバーからも以下のエクスポートが実行できます:

  • SQL — SQL スクリプト出力
  • CSV — CSV ファイル出力
  • Excel — Excel ファイル出力
  • Access — Access データベース出力
  • SQL Server — SQL Server に直接出力
  • ODBC — ODBC 対応データベースに出力

テーブルプロパティ

テーブルを右クリックして プロパティ(&P) を選択すると、 テーブルのカラム定義（カラム名、データ型、長さ等）が表示されます。

テーブル除外

不要なテーブルを一時的に非表示にできます。

  1. テーブル一覧でテーブルを選択します（複数選択可）。
  2. 右クリックメニューの 削除(&D) を選択、 または Delete キーを押します。
  3. 除外されたテーブルはステータスバーに表示されます。

除外を元に戻すには:

  • 編集(&E) > 元に戻す (Ctrl+Z)
  • オブジェクト(&O) > すべてのフィルタをクリア(&F) で全除外を解除
