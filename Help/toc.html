OraDB DUMP Viewer ヘルプ

OraDB DUMP Viewer は、Oracle Data Pump (EXPDP) および従来の EXP 形式のダンプファイルを Oracle クライアントなしで GUI で閲覧・エクスポートするためのスタンドアロンツールです。

目次

  1. はじめに
  2. ダンプファイルを開く
  3. ワークスペース
  4. テーブルプレビュー
  5. 検索・フィルタ
  6. CSV エクスポート
    • Excel エクスポート
    • Access エクスポート
    • SQL Server エクスポート
    • ODBC エクスポート
  7. SQL エクスポート
  8. 一括エクスポート
  9. エクスポートオプション
  10. LOB ファイル抽出
  11. ショートカットキー
  12. 対応形式・制限事項
  13. バージョン情報・ライセンス・エラー報告

動作要件

  • Windows 10 以降 (64bit)
  • .NET 10.0 ランタイム
