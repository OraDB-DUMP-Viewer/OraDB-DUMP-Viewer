检查版本

从菜单选择 帮助(&H) > 关于(&A) 可查看当前版本。

关于对话框会自动与最新的 GitHub 发行版进行比较，并显示是否有可用更新。

  • 如果使用的是最新版本：版本号以绿色显示
  • 如果有更新版本可用：版本号以橙色显示

自动更新

当有新版本可用时，对话框中会出现更新按钮。

  1. 在关于对话框中单击 更新到最新版本 按钮。
  2. 在确认对话框中选择 是。
  3. 最新的 MSI 安装程序将自动下载（显示进度条）。
  4. 下载完成后，应用程序关闭并自动启动安装程序。

您也可以单击 打开发行页面 链接，在浏览器中查看 GitHub 发行页面。

错误报告

如果您发现错误或问题，请通过 帮助(&H) > 错误报告(&R) 进行报告。

以下信息会自动收集并在报告对话框中透明显示：

  • 应用程序版本 / DLL 版本
  • 操作系统版本 / .NET 版本
  • 架构 / 屏幕分辨率 / DPI
  • 转储文件信息（仅大小和格式；不发送路径）

报告通过 Cloudflare Worker 自动提交为 GitHub Issues。 不会收集或发送任何个人信息（用户名、电脑名称、文件路径等）。

如果转储文件小于 50 MB，您可以选择将其附加到报告中。
