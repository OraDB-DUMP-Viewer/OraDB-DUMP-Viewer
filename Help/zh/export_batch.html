批量导出

概述

一次性将多个表导出为 CSV 或 SQL 文件。 每个表输出为一个单独的文件。

操作步骤

  1. 单击工具栏上的 批量导出 按钮。
  2. 显示导出对话框。
  3. 选择输出格式（CSV / SQL）。
  4. 指定输出文件夹。
  5. 确认要导出的表。
    取消勾选不需要的表。
  6. 单击 导出 按钮。

输出文件名

文件名按照 SchemaName.TableName.csv 或 SchemaName.TableName.sql 的格式自动生成。

表的选择/排除

  • 全选 / 取消全选 按钮用于批量操作
  • 反选 用于切换选择状态
  • 单击各个复选框以选择特定的表
提示： 在工作区中被排除的表也会自动从批量导出中排除。

报告输出

通过 对象(&O) > 报告输出(&R)， 您还可以将表信息输出为文本文件。

  • 对象列表 — 输出表名和行数的文本列表
  • 表定义 — 输出所有表的列定义（名称、类型、大小）
