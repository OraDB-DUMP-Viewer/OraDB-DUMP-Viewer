CSV 导出

操作步骤

  1. 在表列表中选择要导出的表。
  2. 单击工具栏上的 CSV 按钮，或选择 对象(&O) > 导出 > 数据(&D)。
  3. 指定目标文件路径。
  4. 开始导出并显示进度对话框。

输出格式

  • 编码：UTF-8（带 BOM）
  • 分隔符：逗号（,）
  • 引号：双引号（"）— 当值包含逗号、换行符或双引号时使用
  • 换行符：CRLF（\r\n）
  • 符合 RFC 4180 标准

选项

以下设置可在导出选项中配置：

  • 标题行 — 是否在第一行输出列名
  • 类型信息行 — 在标题行之后输出列类型信息
  • 日期格式 — DATE/TIMESTAMP 列的输出格式

其他导出格式

Excel 导出

将表数据导出为 Excel 文件（.xlsx / .xls）。

  1. 选择一个表并单击工具栏上的 Excel 按钮。
  2. 指定目标文件路径。
  3. 生成 Excel 文件。

Access 导出

将表数据导出到 Microsoft Access 数据库（.accdb / .mdb）。

  1. 选择一个表并单击工具栏上的 Access 按钮。
  2. 指定目标文件路径。
  3. 在 Access 数据库中创建表并导入数据。

SQL Server 导出

将表数据直接导出到 SQL Server 数据库。

  1. 选择一个表并单击工具栏上的 SQL Server 按钮。
  2. 在连接对话框中输入 SQL Server 连接信息。
  3. 在目标数据库中创建表并插入数据。

如果存在同名表，将自动删除并重新创建。

ODBC 导出

将表数据导出到任何兼容 ODBC 的数据库。

  1. 选择一个表并单击工具栏上的 ODBC 按钮。
  2. 在连接对话框中输入 ODBC 连接字符串或 DSN。
  3. 在目标数据库中创建表并插入数据。

如果存在同名表，将自动删除并重新创建。

现有表的处理

导出目标现有表行为
SQL 脚本DROP TABLE IF EXISTS（可通过复选框切换）
SQL Server自动删除 + 重新创建
ODBC自动删除 + 重新创建
Access重新创建整个文件
Excel覆盖文件
注意： 对于 SQL Server 和 ODBC 导出，必须事先创建目标数据库。
