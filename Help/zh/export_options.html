导出选项

访问方式

单击工具栏上的 选项 按钮以打开导出选项对话框。

日期格式

格式示例说明
YYYY/MM/DD HH:MI:SS2026/01/15 09:30:00标准格式（默认）
YYYYMMDD20260115紧凑格式（仅日期）
YYYYMMDDHH24MISS20260115093000紧凑格式（日期和时间）
自定义任意标记：YYYY, MM, DD, HH24, MI, SS

CSV 选项

设置默认值说明
输出标题行开启在第一行输出列名
输出类型信息行关闭在标题行之后输出列类型

SQL 选项

SQL 导出前会显示一个对话框。 以下选项可通过复选框切换：

设置默认值说明
输出 DROP TABLE + CREATE TABLE开启在 INSERT 语句之前输出 DROP TABLE IF EXISTS 和 CREATE TABLE DDL。可通过复选框切换。删除并重新创建现有表
DBMS 类型OracleOracle / PostgreSQL / MySQL / SQL Server
注意： 选项设置保存在应用程序设置中，重启后仍然有效。
