SQL 导出

操作步骤

  1. 在表列表中选择要导出的表。
  2. 单击工具栏上的 SQL 按钮，或选择 对象(&O) > 导出 > 脚本(&S)。
  3. 指定目标文件路径。
  4. 开始导出。

支持的 DBMS

可为以下 DBMS 生成 INSERT 语句：

DBMS语法差异
OracleTO_DATE 函数，NUMBER 类型原样输出
PostgreSQLCAST 语法，TIMESTAMP 类型
MySQL反引号标识符，DATETIME 类型
SQL Server方括号标识符，DATETIME2 类型

选项

单击 SQL 按钮后，导出前会显示一个对话框。 可配置以下选项：

  • DBMS 类型 — 从 Oracle / PostgreSQL / MySQL / SQL Server 中选择
  • DROP TABLE + CREATE TABLE 输出（默认：开启）— 通过复选框切换。 开启时，在 INSERT 语句之前输出 DROP TABLE IF EXISTS 和 CREATE TABLE DDL。 如果目标数据库中已存在该表，则删除并重新创建。
  • 日期格式 — 字面量格式或函数调用

各 DBMS 的 DROP TABLE 语法

DBMS语法
Oracle / PostgreSQLDROP TABLE IF EXISTS "table_name" CASCADE;
MySQLDROP TABLE IF EXISTS `table_name`;
SQL ServerIF OBJECT_ID('table_name', 'U') IS NOT NULL DROP TABLE [table_name];
注意： LOB 数据（BLOB/CLOB）不包含在 SQL 导出中。 请使用 LOB 文件提取来导出 LOB 数据。
