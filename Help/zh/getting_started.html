什么是 OraDB DUMP Viewer？

OraDB DUMP Viewer 是一款独立工具，可在无需 Oracle 客户端的情况下浏览和导出 Oracle 数据库转储文件（EXP / EXPDP）。

主要功能

  • 转储文件分析 — 支持 EXP（旧版）和 EXPDP（Data Pump）格式
  • 表列表 — 以树形视图显示模式、表名和列信息
  • 数据预览 — 分页显示表数据，支持排序和复制
  • 高级搜索 — 使用 12 种运算符和 AND/OR 复合条件筛选数据
  • CSV 导出 — 将表数据导出为 CSV 文件
  • SQL 导出 — 生成 INSERT 语句（Oracle / PostgreSQL / MySQL / SQL Server）
  • Excel 导出 — 导出为 .xlsx / .xls 格式
  • Access 导出 — 导出到 .mdb / .accdb 数据库
  • SQL Server 导出 — 直接导出到 SQL Server 数据库
  • ODBC 导出 — 导出到任何兼容 ODBC 的数据库
  • 批量导出 — 一次性将多个表导出为 CSV/SQL
  • LOB 文件提取 — 将 BLOB/CLOB/NCLOB 数据提取为单独的文件
  • 工作区 — 保存和恢复工作状态
  • 表排除 — 隐藏不需要的表（支持撤销/重做）
  • 多语言支持 — 支持 10 种语言，包括日语、英语、中文、韩语、德语、法语、西班牙语、意大利语、俄语和葡萄牙语（巴西）
  • 错误报告 — 可直接从应用程序内向 GitHub Issues 报告错误
  • 自动更新 — 检测新版本并提供一键更新

支持的转储格式

格式工具支持状态
EXP（旧版）exp / imp支持（Oracle 7–11g）
EXPDP（Data Pump）expdp / impdp支持（Oracle 10g–23ai）
EXPDP 压缩expdp COMPRESSION=ALL部分支持（仅表定义）

安装

  1. 运行分发的 MSI 安装程序。
  2. 按照安装向导的指示操作。
  3. 从桌面或开始菜单启动"OraDB DUMP Viewer"。
