LOB 文件提取

概述

将表中 BLOB / CLOB / NCLOB 列存储的数据按每条记录提取为一个文件。

支持的数据类型

列类型输出内容
BLOB二进制文件（图片、PDF、ZIP 等）
CLOB文本文件（使用转储文件的字符集进行解码）
NCLOBUnicode 文本文件

操作步骤

  1. 在表列表中选择包含 LOB 列的表。
  2. 从菜单选择 文件(&F) > 提取文件(&F)。
  3. 显示 LOB 提取对话框。
  4. 配置以下项目：
    • LOB 列 — 选择要提取的 LOB 列
    • 输出文件夹 — 输出文件的目录
    • 文件命名 — 顺序编号或列值
    • 扩展名 — 输出文件的扩展名（默认：lob）
  5. 单击 提取 按钮。

文件命名方式

方式示例说明
顺序编号000001.jpg、000002.jpg从 1 开始的 6 位顺序编号
列值filename_col_value.jpg使用指定列的值作为文件名

使用示例

提取图片文件

当 JPEG 图片存储在 BLOB 列中时：

  1. LOB 列：选择 IMAGE_DATA (BLOB)
  2. 扩展名：jpg
  3. 文件命名：顺序编号或 ID 列值

提取文本数据

当 XML 或 JSON 存储在 CLOB 列中时：

  1. LOB 列：选择 XML_CONTENT (CLOB)
  2. 扩展名：xml
注意： 当 LOB 记录较多时，提取可能需要较长时间。 可通过进度对话框取消操作。
说明： 在表预览中，LOB 数据显示为占位符： %BLOB%、%CLOB%、%NCLOB%。 请使用此 LOB 文件提取功能获取实际数据。
