打开转储文件

操作步骤

  1. 从菜单选择 文件(&F) > 打开转储文件(&O)...。
    或单击工具栏上的 打开 按钮。
  2. 在文件对话框中，选择一个转储文件（.dmp）。
  3. 开始分析并显示进度条。
  4. 完成后，表列表将显示在工作区中。

自动检测

转储文件格式（EXP / EXPDP / EXPDP 压缩）通过分析文件的前导字节和标记字节来自动检测。

格式检测方法
EXP前导字节 0x01-0x05 + "EXPORT:" 字符串
EXPDP"xml version" 字符串
EXPDP（压缩）"KGC" + "HDR" 标记

字符集

转储文件的字符集也会自动检测。 支持的字符集：

  • AL32UTF8 / UTF8
  • JA16SJIS (Shift_JIS)
  • JA16EUC (EUC-JP)
  • AL16UTF16
  • US7ASCII
  • WE8ISO8859P1 / WE8MSWIN1252
注意： 对于压缩的 EXPDP 转储文件，只能读取表定义（DDL）。 不支持数据记录的解压。
