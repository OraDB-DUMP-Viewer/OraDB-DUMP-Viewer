表预览

打开表

在表列表中双击表名以打开数据预览。 或右键单击表并选择 打开(&O)。

数据显示

数据在 DataGridView 中显示。

  • 列名显示在列标题中。
  • 每行显示一条记录。
  • NULL 值显示为空白。
  • LOB 数据（BLOB/CLOB/NCLOB）显示为 %BLOB%、%CLOB%、%NCLOB%。

分页

当数据量较大时，将分页显示。

  • 每页行数 — 可通过右上角的数字输入框调整（默认：100 行）
  • 页面信息 — 显示当前页码和总页数
  • 使用导航按钮在页面之间切换

排序

单击列标题可按该列排序数据。 再次单击可在升序和降序之间切换。

复制单元格

在预览中选择单元格，按 Ctrl+C 复制到剪贴板。 也支持多单元格选择和复制。

高级搜索

单击预览画面上的 高级搜索 按钮，可使用 12 种运算符和 AND/OR 复合条件筛选数据。 详细信息请参阅 搜索与筛选。

从预览导出

可从预览工具栏进行以下导出：

  • SQL — SQL 脚本输出
  • CSV — CSV 文件输出
  • Excel — Excel 文件输出
  • Access — Access 数据库输出
  • SQL Server — 直接输出到 SQL Server
  • ODBC — 输出到兼容 ODBC 的数据库

表属性

右键单击表并选择 属性(&P) 可查看列定义（列名、数据类型、长度等）。

表排除

您可以临时隐藏不需要的表。

  1. 在表列表中选择表（支持多选）。
  2. 从右键菜单选择 删除(&D)，或按 Delete 键。
  3. 被排除的表会显示在状态栏中。

撤销排除：

  • 编辑(&E) > 撤销（Ctrl+Z）
  • 对象(&O) > 清除所有筛选(&F) 可移除所有排除
