■ ソフトウェア概要

  Oracle Data Pump (EXPDP) および Export (EXP) で出力したダンプファイル
  (.dmp) の中身を、Oracle 環境なしで GUI で閲覧・エクスポートできる
  Windows デスクトップアプリケーションです。

  主な機能:
  - テーブル一覧の表示（パーティションテーブル対応）
  - データプレビュー（ページング・検索機能付き）
  - CSV / SQL / Excel / Access / SQL Server / ODBC へのエクスポート
  - LOB データ（BLOB/CLOB）の抽出
  - ワークスペースによるセッション管理
  - 10言語対応（日本語・英語・中国語・韓国語・ドイツ語・フランス語・
    スペイン語・イタリア語・ロシア語・ポルトガル語）

■ インストール方法

  【インストーラー版】
  OraDBDumpViewer_vX.X.X_installer_x64.exe (または arm64) を実行し、
  画面の指示に従ってインストールしてください。
  既定のインストール先: C:\Program Files\OraDB DUMP Viewer

  【ポータブル版 (ZIP)】
  ZIP ファイルを任意のフォルダに展開してください。
  展開先の「OraDB DUMP Viewer.exe」をそのまま実行できます。

  【winget】
  winget install OraDBDumpViewer.OraDBDumpViewer

■ アンインストール方法

  【インストーラー版】
  Windows の「設定」→「アプリ」→「インストールされているアプリ」から
  「OraDB DUMP Viewer」を選択し、アンインストールしてください。

  【ポータブル版】
  展開したフォルダごと削除してください。
  ユーザー設定を削除する場合は、以下のフォルダも削除してください:
  %APPDATA%\OraDBDUMPViewer
